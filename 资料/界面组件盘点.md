# UI Inventory（界面组件盘点）

---

## Buttons

### 按钮类型（5 种）

| 类型 | CSS 类 | 出现页面 | 用途 |
|------|--------|----------|------|
| Primary | `.btn-primary` | 所有页面 | 主要操作：查询、保存、新增、登录、分配、录入跟进 |
| Secondary | `.btn`（默认） | 所有页面 | 次要操作：取消、重置、刷新、返回、编辑 |
| Danger | `.btn-danger` | 线索列表、客户列表、站点管理、用户管理、账号绑定、登录安全 | 危险操作：回收、删除、停用、解绑、退出登录 |
| Text | `.btn-text` | 所有表格页、详情页 | 表格行内操作：查看详情、编辑、跟进、打标、分配、删除 |
| Icon Button | `.icon-btn` | 应用框架 | 全局操作：关闭标签页、关闭当前标签 |

### 按钮状态
- 默认
- 禁用（`disabled` 属性，opacity 0.5）
- 悬停（边框色/文字色变为主色）

### 使用模式
- 按钮带图标（通过 `iconName` 参数）：新增(plus)、刷新(refresh)、导出(export)、查询(search)等
- 按钮权限控制：通过 `permission` 参数控制可见性

## Inputs

### 输入框类型（5 种）

| 类型 | 出现页面 | 用途 |
|------|----------|------|
| 文本输入框（`input[type="text"]`） | 所有列表页搜索面板、表单页 | 关键词搜索、表单字段录入 |
| 密码输入框（`input[type="password"]`） | 登录页、修改密码页 | 密码输入 |
| 日期输入框（`input[type="date"]`） | 搜索面板、表单页 | 日期范围筛选、日期选择 |
| 数字输入框（`input[type="number"]`） | 合同中心表单 | 合同金额输入 |
| 复选框（`input[type="checkbox"]`） | 表格行选择、角色权限页、登录页 | 多选、权限分配、记住密码 |
| 文本域（`textarea`） | 跟进记录、回收/分配弹窗 | 多文本输入 |

### 统一样式
- 最小高度：34px
- 边框：1px solid #d9d9d9
- 圆角：6px
- Focus 态：蓝色边框 + 2px 蓝色阴影

## Form

### 表单组件（2 种）

| 类型 | 出现页面 | 用途 |
|------|----------|------|
| `simpleForm` | 所有 Drawer 弹窗、Modal 弹窗 | 标准数据录入表单 |
| `searchPanel` | 所有列表页 | 筛选条件区域 |

### simpleForm 规格
- 两列网格布局（`.form-grid`，`repeat(2, 1fr)`）

### searchPanel 规格
- 四列网格布局（`.search-grid`，`repeat(4, 1fr)`）
- 下方查询 + 重置按钮

## Tags

### Tag 类型（6 种）

| 类型 | CSS 类 | 外观 | 出现页面 | 用途 |
|------|--------|------|----------|------|
| 默认 | `.tag` | 灰色 | 所有页面 | 一般标签、信息来源 |
| Primary | `.tag-primary` | 蓝色 | 线索状态、阶段标记 | 跟进中、高意向、主要联系人 |
| Success | `.tag-success` | 绿色 | 所有页面 | 启用、正常、成功、已签约 |
| Danger | `.tag-danger` | 红色 | 所有页面 | 停用、异常、失败、无效、丢失 |
| Warning | `.tag-warning` | 橙色 | 线索详情、邮件 | 待确认、公海待分配 |
| AI | `.tag-ai` | 紫色 | 邮件、WhatsApp、线索详情 | AI 自动生成的标签 |

### 自动识别规则
- Tag 自动根据文字内容匹配类型
- `AI:` 前缀自动识别为 AI Tag
- 匹配特定关键词（启用、停用、成功、失败等）自动变色

## Empty State

### 空状态组件

**出现页面/场景：**

| 使用场景 | 用途 |
|----------|------|
| 邮件页面 - 无邮件 | 暂无邮件 |
| 线索列表 - 无符合条件数据 | 暂无数据 |
| 线索详情 - 无跟进记录 | 暂无跟进记录 |

| 页面不存在 | 页面不存在 |
| 无权限页面 | 暂无访问权限 |

### 空状态类型
- `emptyState`：业务空数据，含标题、说明、可选操作按钮
- `noPermission`：权限拒绝，独立视觉样式（红色图标）
- `errorState`：加载失败，含重新加载按钮

### 统一要素
- 居中文字 + 图标
- 业务引导说明
- 操作入口

### 需要统一设计的组件（图中 ✅ 列）
所有 30 种组件都建议统一设计，因为它们分布在多个页面中复用。

### 建议保持一致的组件（图中 ✅ 列）
- **高优先级保持一致**：Button、Input、Select、dataTable、Tag、Modal、Drawer、Toast、Pagination、Avatar、Breadcrumb

### 不需要额外设计的组件
- 无权限页、错误状态页、系统状态页（专用页面，使用通用状态组件即可）
