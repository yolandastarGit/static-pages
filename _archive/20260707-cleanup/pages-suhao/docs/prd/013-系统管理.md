## 一、模块概述

**模块归属：** 系统管理（一级菜单）

**模块定位：** 提供平台级配置与管理能力，是管理员对系统进行集中管控的统一入口。包含用户/角色/菜单/字典/参数/系统参数/平台级业务配置/系统日志共 12 个二级菜单。

**面向角色：** 系统管理员

**菜单结构：**

```
├── 系统管理
│   ├── 用户管理
│   ├── 角色管理
│   ├── 菜单管理
│   ├── 字典管理
│   ├── 参数设置
│   ├── 系统参数
│   ├── 消息推送配置
│   ├── 邮件服务配置
│   ├── WhatsApp配置
│   ├── 系统开关
│   └── 系统日志
```

## 二、用户管理

### 2.1 功能定位

管理后台所有用户账号，包括新增、编辑、启用/停用、重置密码、分配角色、分配站点、绑定钉钉账号。

### 2.2 页面布局

用户管理页面采用 **顶部搜索 + 表格列表** 布局：

| 区域 | 内容 |
| :--- | :--- |
| **搜索栏** | 关键词输入 + 【查询】/【重置】按钮 + 【新增用户】按钮 |
| **表格区** | 用户列表 + 分页 |

### 2.3 搜索筛选项

| 筛选项 | 组件类型 | 说明 |
| :--- | :--- | :--- |
| 关键词 | 文本输入 | 模糊匹配登录账号/姓名/手机号/邮箱 |
| 状态 | 下拉选择 | 全部 / 启用 / 停用 |
| 角色 | 下拉选择 | 按角色筛选 |
| 所属站点 | 下拉选择 | 按授权站点筛选 |

### 2.4 表格字段

| 字段 | 说明 |
| :--- | :--- |
| 姓名 | 用户姓名 |
| 登录账号 | 系统登录账号（唯一） |
| 手机号 | 用户手机号 |
| 邮箱 | 用户邮箱 |
| 角色 | 所属角色 |
| 所属站点 | 授权的站点 |
| 状态 | 启用 / 停用 |
| 操作 | 编辑 / 启用·停用 / 重置密码 |

### 2.5 新增/编辑用户弹窗

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :---: | :--- |
| 姓名 | 文本输入 | 是 | 用户姓名 |
| 登录账号 | 文本输入 | 是 | 唯一，不可重复 |
| 手机号 | 文本输入 | 否 | 用户手机号 |
| 邮箱 | 文本输入 | 否 | 用户邮箱 |
| 角色 | 下拉选择 | 是 | 从角色列表中选择 |
| 所属站点 | 下拉选择（多选） | 是 | 可授权多个站点 |
| 绑定钉钉账号 | 下拉选择 | 否 | 选择已同步的钉钉用户，用于钉钉扫码登录 |
| 状态 | 开关 | 是 | 启用 / 停用 |

### 2.6 操作流程

#### 2.6.1 新增用户

1. 点击【新增用户】按钮 → 弹出新增弹窗
2. 填写字段，点击【保存】
3. 系统自动生成初始密码（默认 123456），首次登录强制修改
4. 列表新增该用户

**校验：**
- 登录账号唯一
- 手机号格式校验
- 邮箱格式校验
- 至少选择 1 个角色

#### 2.6.2 编辑用户

1. 点击操作列【编辑】→ 弹出编辑弹窗
2. 修改字段（**登录账号不可修改**），点击【保存】
3. 列表更新

#### 2.6.3 重置密码

1. 点击操作列【重置密码】
2. 二次确认：弹窗提示"重置后该用户密码将恢复为初始密码（123456），首次登录将强制修改"
3. 确认后密码重置
4. Toast 提示"密码已重置"

#### 2.6.4 启用/停用

**停用用户规则：**
- 点击操作列【停用】→ 二次确认 → 停用生效
- 停用后，该用户**无法登录**系统
- 名下数据（线索/客户/合同）**同步停用**（按状态字段标记为"无效"或保留只读状态），**不进行数据转移**
- 已停用用户仍可在列表展示（状态标签为"停用"），允许后续重新启用

**启用用户规则：**
- 点击操作列【启用】→ 直接生效
- 启用后该用户恢复登录
- 名下数据状态自动恢复

### 2.7 角色配置快捷入口

用户管理页面提供"角色管理"快捷入口，点击跳转至「系统管理 → 角色管理」页面。


## 三、角色管理

### 3.1 功能定位

管理角色列表，并为角色配置菜单权限、按钮权限、数据权限。

### 3.2 页面布局

角色管理页面采用 **左侧角色列表 + 右侧权限配置** 布局：

| 区域 | 宽度 | 内容 |
| :--- | :--- | :--- |
| **左栏（角色列表）** | 240px | 角色列表 + 新增/删除角色按钮 |
| **右栏（权限配置）** | 自适应 | 菜单权限、按钮权限、数据权限配置 |

### 3.3 操作流程

#### 3.3.1 新增角色

1. 左栏顶部【新增角色】按钮 → 弹出新增弹窗
2. 填写：角色名称、角色编码、描述
3. 点击【保存】→ 角色创建成功，进入右栏配置权限

#### 3.3.2 编辑角色

1. 左栏选中角色 → 点击【编辑】按钮
2. 修改基本信息，点击【保存】

#### 3.3.3 删除角色

1. 左栏选中角色 → 点击【删除】按钮
2. 二次确认：弹窗提示"删除后该角色的所有用户将被解除关联"
3. 确认后删除

**校验：**
- 超级管理员角色不可删除
- 当前有用户关联的角色不可删除（提示"该角色下还有 N 个用户，请先解除关联"）

### 3.4 权限配置

#### 3.4.1 菜单权限

| 维度 | 说明 |
| :--- | :--- |
| 配置方式 | 树形勾选（全选/取消全选） |
| 作用域 | 控制左侧导航菜单的可见性 |
| 默认值 | 空（无任何菜单权限） |

#### 3.4.2 按钮权限

| 维度 | 说明 |
| :--- | :--- |
| 配置方式 | 按模块分组勾选 |
| 作用域 | 控制页面内操作按钮的可见性 |
| 默认值 | 空（无任何按钮权限） |

#### 3.4.3 数据权限

| 粒度 | 含义 | 应用范围 |
| :--- | :--- | :--- |
| 仅本人 | 仅可查看自己作为负责人的数据 | 线索 / 客户 / 合同 |
| 本团队 | 可查看所属团队所有业务员的数据 | 线索 / 客户 / 合同 |
| 本区域 | 可查看所属区域所有业务员的数据 | 线索 / 客户 / 合同 |
| 全部 | 可查看全公司所有业务员的数据 | 线索 / 客户 / 合同 |

**数据权限规则：**
- 业务模块（线索/客户/合同）均应用数据权限粒度
- 合同数据权限**不再按状态细分**（仅按粒度 4 档）
- 线索/客户/合同模块共用同一数据权限配置

### 3.5 内置角色保护

- 超级管理员（系统管理员）不可停用、不可删除、不可降权
- 系统必须至少保留一个启用且具备系统管理权限的管理员用户

### 3.6 角色继承

MVP 阶段不提供角色继承功能。角色的权限独立配置，不支持父子角色关系。


## 四、菜单管理

### 4.1 功能定位

管理系统侧边栏菜单、页面入口、按钮级权限点，是角色配置中"菜单权限"和"按钮权限"的基础数据来源。

### 4.2 页面布局

菜单管理采用 **树形结构 + 右侧详情** 布局。

### 4.3 菜单节点字段

| 字段 | 说明 |
| :--- | :--- |
| 菜单名称 | 展示在侧边栏的菜单文字 |
| 菜单图标 | 菜单对应的图标 |
| 路由路径 | 菜单对应的前端路由 |
| 菜单层级 | 一级 / 二级 |
| 父级菜单 | 二级菜单的父级 |
| 排序 | 决定菜单显示顺序 |
| 状态 | 启用 / 停用 |
| 权限编码 | 用于按钮权限校验的编码（系统内唯一） |

### 4.4 操作流程

#### 4.4.1 新增菜单

1. 选中父级节点 → 点击【新增子菜单】→ 弹出新增弹窗
2. 填写字段，点击【保存】
3. 树形列表新增该节点

**校验：**
- 权限编码全局唯一
- 路由路径不能重复
- 一级菜单不允许设置父级

#### 4.4.2 编辑菜单

1. 选中节点 → 点击【编辑】→ 弹出编辑弹窗
2. 修改字段，点击【保存】

#### 4.4.3 删除菜单

1. 选中节点 → 点击【删除】→ 二次确认
2. 含子菜单的父节点不允许删除（提示"请先删除子菜单"）

#### 4.4.4 拖拽排序

- 鼠标拖拽菜单节点调整顺序
- 仅允许同级拖拽
- 拖拽后自动保存

#### 4.4.5 启用/停用

**重要规则：菜单停用后**

- 该菜单在**所有用户的侧边栏入口不可见**
- **但已配角色的"菜单权限"配置仍保留**（仅入口隐藏，权限点不回收）
- 重新启用后，入口恢复
- 不可停用当前用户所在角色的所有菜单（防止用户失去全部入口）

### 4.5 按钮权限点管理

| 元素 | 说明 |
| :--- | :--- |
| 权限点列表 | 当前菜单下所有按钮级权限点 |
| 权限点编码 | 唯一编码（菜单权限编码 + `:` + 按钮标识），全局唯一 |
| 权限点名称 | 业务语义名称（如"新增用户""导出数据"） |


## 五、字典管理

### 5.1 功能定位

管理系统枚举与业务字典，为业务模块提供统一的下拉选项数据源。

### 5.2 页面布局

字典管理采用 **左侧字典分类 + 右侧字典项** 布局。

### 5.3 字典分类

字典分类按业务域组织，每个分类下维护一组字典项：

| 分类 | 典型字典项 |
| :--- | :--- |
| 客户域 | 行业、国家/地区、客户状态、来源渠道 |
| 线索域 | 线索状态、跟进方式、跟进阶段、标签类型 |
| 合同域 | 合同状态、合同类型、币种 |
| 系统域 | 登录方式、操作类型 |

### 5.4 字典项字段

| 字段 | 说明 |
| :--- | :--- |
| 字典编码 | 字典项唯一编码 |
| 字典名称 | 业务展示名称 |
| 排序 | 显示顺序（数值越小越靠前） |
| 状态 | 启用 / 停用 |
| 备注 | 备注说明 |

### 5.5 操作流程

#### 5.5.1 新增字典项

1. 选中字典分类 → 点击【新增字典项】→ 弹出新增弹窗
2. 填写字段，点击【保存】

**校验：**
- 字典编码在分类下唯一
- 字典名称不可为空

#### 5.5.2 编辑字典项

1. 选中字典项 → 点击【编辑】→ 修改字段 → 保存

#### 5.5.3 删除字典项

1. 选中字典项 → 点击【删除】→ 二次确认

**校验：**
- 已被业务数据引用的字典项不允许删除（提示"该字典项已被 N 条业务数据引用，不可删除"）
- 仅可停用

#### 5.5.4 启用/停用

**重要规则：字典项停用后**

- **新增数据**不可再选择该字典项（下拉中隐藏）
- **存量历史数据**展示不受影响（已用此字典项的历史业务数据仍正常显示该值）
- 重新启用后，选项恢复


## 六、参数设置

### 6.1 功能定位

维护系统通用业务参数，配置影响全局行为的参数项。

### 6.2 参数列表

| 参数名称 | 参数键 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| 登录超时时长 | session_timeout | 30 分钟 | 用户无操作自动退出登录的时长 |
| 密码最小长度 | password_min_length | 8 位 | 密码长度最小值 |
| 密码复杂度 | password_complexity | 数字+大小写字母 | 密码字符类型要求 |
| 列表默认每页条数 | list_page_size | 20 条 | 列表页默认每页条数 |
| 单次导出上限 | export_max_rows | 10000 条 | 列表导出最大行数限制 |
| 首次登录强制改密 | first_login_change_password | 开启 | 新建/重置密码后首次登录是否强制改密 |
| 单端登录限制 | single_login | 关闭 | 是否限制同一账号同时只能在一处登录 |
| 会话刷新间隔 | session_refresh_interval | 5 分钟 | Token 续期间隔 |
| 个人目标金额 | personal_target_amount | 0（无目标） | 销售业务员的销售目标金额（系统级默认值，0 表示无目标） |

### 6.3 操作流程

#### 6.3.1 编辑参数

1. 参数列表 → 点击操作列【编辑】→ 弹出编辑弹窗
2. 修改参数值（不同参数类型对应不同组件：文本/数字/开关/下拉）
3. 点击【保存】→ 参数立即生效

#### 6.3.2 生效时间

- 大多数参数**立即生效**
- `session_timeout` / `session_refresh_interval` / `first_login_change_password` 在**用户下次登录**时生效
- 参数变更记录到系统日志

#### 6.3.3 修改权限

- 仅系统管理员可修改
- 修改操作记录到系统日志


## 七、系统参数

### 7.1 功能定位

维护系统级业务规则参数，是业务模块运行时依赖的核心规则。

### 7.2 业务规则配置

| 规则名称 | 规则说明 | 当前配置 | 操作 |
| :--- | :--- | :--- | :--- |
| 有效询盘自动入池 | 站点接口、网站表单等渠道识别的有效询盘，自动进入公海池等待分配 | 开启 | 编辑 |
| 采集邮箱来件自动创建线索 | 运营邮箱收到来件后，按解析规则生成线索或进入待确认队列 | 开启 | 编辑 |
| 陌生发件人自动创建线索 | 发件人未匹配现有客户/联系人时，是否自动创建新线索 | 关闭 | 编辑 |
| 未跟进自动回收天数 | 业务员负责线索在期限内无任何跟进记录时，系统自动回收至公海池 | 14 天 | 编辑 |
| 客户负责人变更是否同步线索负责人 | 客户转移或负责人调整时，关联线索负责人是否一并更新 | 是 | 编辑 |
| 新增合同自动归属客户负责人 | 录入或关联合同时，默认将合同业务员设为客户当前负责人 | 是 | 编辑 |
| 渠道定义规则 | 定义获客分析中各渠道的分类与映射（邮件/WhatsApp） | 系统默认 | 编辑 |

### 7.3 规则编辑弹窗

| 字段 | 组件类型 | 说明 |
| :--- | :--- | :--- |
| 规则名称 | 只读展示 | 规则名称 |
| 规则说明 | 只读展示 | 规则说明 |
| 当前配置 | 开关 / 数字输入 / 下拉选择 | 根据规则类型动态变化 |

### 7.4 字段差异说明

| 规则类型 | 配置组件 | 可选值 |
| :--- | :--- | :--- |
| 开关型（自动入池/自动创建线索等） | 开关 | 开启 / 关闭 |
| 数值型（自动回收天数） | 数字输入 | 1-365 天 |
| 枚举型（渠道定义规则） | 下拉选择 | 系统默认 / 自定义映射 |
| 联动型（客户负责人变更） | 开关 | 是 / 否 |

## 八、消息推送配置

### 8.1 功能定位

配置系统消息推送的通道与基础参数，支持钉钉、站内信等渠道接入。

### 8.2 配置项

| 配置项 | 组件类型 | 说明 |
| :--- | :--- | :--- |
| 推送渠道 | 多选下拉 | 钉钉 / 站内信（按需启用） |
| 钉钉应用 AppKey | 文本输入 | 钉钉开放平台应用凭证 |
| 钉钉应用 AppSecret | 密码输入 | 钉钉开放平台应用密钥 |
| 钉钉机器人 Webhook | 文本输入 | 群机器人推送地址 |
| 站内信保留天数 | 数字输入 | 站内信消息保留时长 |

### 8.3 操作流程

#### 8.3.1 保存配置

1. 顶部表单填写/修改各配置项
2. 点击底部【保存】按钮 → 配置立即生效

#### 8.3.2 测试推送

1. 点击底部【测试推送】按钮 → 弹出测试弹窗
2. 填写：测试用户（选择）、测试内容
3. 点击【发送测试】→ 系统调用渠道发送测试消息
4. 结果展示：
   - 成功：弹窗提示"测试推送成功，请到目标账号确认"
   - 失败：弹窗提示具体失败原因（认证失败/网络超时/参数错误/渠道未启用）

#### 8.3.3 多渠道优先级

钉钉与站内信可同时启用，**推送时同时送达**。无优先级区分。接收人可在个人中心选择是否屏蔽某渠道。

## 十三、系统级状态页

> **职责说明：** 本节定义系统级兜底页面的统一规范。状态页不作为独立菜单存在，而是在用户访问不可达页面或系统异常时自动展示。

### 13.1 状态页清单

| 状态页 | 触发场景 | 路由 | 展示内容 |
| :--- | :--- | :--- | :--- |
| 404 | 访问不存在的页面或资源 | /404 | 页面未找到提示 + 返回首页按钮 |
| 403 无权限 | 用户无访问权限 | /403 | 无权限提示 + 返回首页按钮 |
| 500 | 系统内部错误 | /500 | 系统繁忙提示 + 重新加载按钮 |
| Loading | 全局加载中 | — | 骨架屏 / Loading 动画 |
| 维护中 | 系统维护模式开启 | /maintenance | 维护中提示 + 预计恢复时间 |
| 网络异常 | 网络断开 | — | 网络异常提示 + 重新连接按钮 |

### 13.2 404 页面

**触发场景：**
- 访问不存在的 URL
- 访问已删除/已停用的菜单对应路由
- 资源被永久删除

**展示要素：**
- 大型 404 数字插图
- 提示文案："抱歉，您访问的页面不存在"
- 帮助说明："请检查 URL 是否正确，或返回工作台"
- 【返回首页】按钮（跳转 /dashboard）
- 【返回上一页】按钮

### 13.3 403 无权限页面

**触发场景：**
- 用户无菜单权限
- 用户无按钮权限（操作拦截）
- 角色被降权导致无访问权限

**展示要素：**
- 403 插图
- 提示文案："抱歉，您没有访问此页面的权限"
- 帮助说明："如需访问，请联系管理员开通权限"
- 【返回首页】按钮
- 隐藏【返回上一页】按钮（避免循环跳转）

### 13.4 500 错误页面

**触发场景：**
- 服务端 5xx 错误
- 未捕获的运行时异常

**展示要素：**
- 500 插图
- 提示文案："抱歉，系统繁忙，请稍后再试"
- 帮助说明：可展示错误编号（用于技术支持定位）
- 【重新加载】按钮
- 【联系管理员】按钮（弹窗展示管理员联系方式）

### 13.5 Loading 态

**触发场景：**
- 首次页面加载
- 大数据量查询
- 全局路由切换

**展示规范：**
- 列表/表格：骨架屏（5 行骨架行）
- 表单：表单骨架（占位元素）
- 全局加载：顶部进度条（不阻挡操作）

### 13.6 维护中页面

**触发场景：**
- 系统管理 → 系统开关 → 维护模式 开启

**展示要素：**
- 维护插图
- 提示文案："系统维护中，请稍后再试"
- 预计恢复时间（管理员可在维护模式开启时配置）
- 不提供任何操作按钮（非管理员直接拒绝访问）

### 13.7 网络异常

**触发场景：**
- 客户端断网
- 服务端不可达

**展示要素：**
- 网络异常插图
- 提示文案："网络连接失败，请检查网络"
- 【重新连接】按钮
