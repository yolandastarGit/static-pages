##  一、模块概述

**模块归属：** 分析中心 → 客户经营

**模块定位：** 提供客户维度的经营数据分析，帮助管理者了解客户总量、增长趋势、行业分布、地域分布及线索转化效率。

**面向角色：** 销售主管、区域负责人


## 二、页面整体布局

客户经营页面采用 **顶部筛选栏 + 指标卡 + 图表区** 布局：

| 区域 | 内容 |
| :--- | :--- |
| **顶部筛选栏** | 时间范围选择器 + 站点筛选（管理员/区域负责人可见） |
| **指标卡区** | 4 个核心指标卡片 |
| **图表区** | 客户行业分布（饼图）+ 客户国家分布（柱状图/地图）+ 年度转化趋势（折线图） |


## 三、顶部筛选栏

**功能定位：** 统一控制页面所有图表的数据范围。

| 筛选项 | 类型 | 说明 | 默认值 |
| :--- | :--- | :--- | :--- |
| **时间范围** | 日期区间 | 控制页面所有图表的数据统计范围 | 本年（本年 1 月 1 日至今日） |
| **站点筛选** | 下拉选择 | 按站点过滤数据，仅管理员/区域负责人可见 | 全部站点 |


## 四、指标卡区

位于筛选栏下方，一行展示 4 个核心指标卡片。

### 卡片一：客户总数

**定义：** 截至当前时间，系统中的客户总数量。

**展示内容：**

| 元素 | 说明 |
| :--- | :--- |
| 主数值 | 客户总数（如 12,840） |
| 变化趋势 | 较上一统计周期（如上月）的增长率，格式：+X.X% |

### 卡片二：新增客户（本月）

**定义：** 在当前统计周期内（默认本月），新创建的客户数量。

**展示内容：**

| 元素 | 说明 |
| :--- | :--- |
| 主数值 | 新增客户数（如 846） |
| 变化趋势 | 较上一统计周期（如上月）的增长率，格式：+/-X.X% |

### 卡片三：成交客户

**定义：** 在当前统计周期内（默认本月），有签约合同的客户数量（去重计数，一个客户多份合同计为 1）。

**展示内容：**

| 元素 | 说明 |
| :--- | :--- |
| 主数值 | 成交客户数（如 3,124） |
| 变化趋势 | 较上一统计周期（如上月）的增长率，格式：+/-X.X% |

### 卡片四：客户增长率

**定义：** 客户数量的增长速度。

**计算公式：** `（本月客户总数 - 上月客户总数）/ 上月客户总数 × 100%`

**展示内容：**

| 元素 | 说明 |
| :--- | :--- |
| 主数值 | 增长率百分比（如 24.5%） |
| 趋势描述 | 辅助文案，如“稳定增长” |


## 五、图表区

### 5.1 客户行业分布

**图表类型：** 饼图

**功能定位：** 展示客户按行业的分布占比，帮助了解客户构成。

**数据来源：** 客户基础信息中的“行业”字段

**展示规则：**
- 按客户数量统计各行业占比
- 展示行业名称 + 占比百分比
- 默认展示前 5 大行业，其余合并为“其他”
- 示例：信息技术（IT）45.2%、金融服务 22.8%、制造业 18.5%、其他 13.5%


### 5.2 客户国家分布

**图表类型：** 柱状图 或 世界地图

**功能定位：** 展示客户按国家/地区的分布情况，了解市场覆盖。

**数据来源：** 客户基础信息中的“国家/地区”字段

**展示规则：**
- 按客户数量统计各国客户数
- 柱状图：X 轴为国家名称，Y 轴为客户数量，按数量降序排列
- 默认展示前 10 个国家，其余合并为“其他”


### 5.3 年度转化趋势（线索→客户转化率）

**图表类型：** 折线图

**功能定位：** 展示线索转化为客户的转化率月度变化趋势，帮助评估销售漏斗效率。

**数据维度：** 仅展示“线索→客户转化率”月度变化趋势

**转化率计算：** `当月线索转客户数 / 当月有效线索数 × 100%`

**展示规则：**
- X 轴：月份（本年 1 月至当前月）
- Y 轴：转化率百分比
- 单条折线展示转化率趋势
- 数据点标注百分比数值
- 示例：1 月 12.4%、3 月 15.2%、6 月 18.7%


## 六、图表交互说明

| 交互行为 | 说明 |
| :--- | :--- |
| 饼图悬停 | 显示该分类的客户数量和占比 |
| 柱状图悬停 | 显示该国家的客户数量 |
| 趋势图悬停 | 显示该月份的转化率数值 |
| 时间范围切换 | 切换后所有图表数据同步刷新 |
| 站点切换 | 切换后所有图表按站点过滤 |


## 七、数据更新频率

| 数据范围 | 说明 |
| :--- | :--- |
| 指标卡数据 | 截至当前时间的实时数据 |
| 图表数据 | 按所选时间范围实时统计 |


## 八、空态与异常态

| 场景 | 处理方式 |
| :--- | :--- |
| 无任何客户数据 | 各指标卡主数值显示"--"，图表显示空状态占位图 |
| 饼图/柱状图无数据 | 图表区域显示“暂无分布数据” |
| 地图无数据 | 地图区域显示“暂无国家分布数据” |
| 趋势图无数据 | 坐标系保留，折线区域显示“暂无转化数据” |
| 部分国家/行业无数据 | 图表中仅展示有数据的分类，无数据时不显示 |
| 数据加载中 | 图表区域显示骨架屏 |
| 数据加载失败 | 显示“数据加载失败，请刷新重试”，提供【重新加载】按钮 |


## 九、角色数据权限

| 角色 | 数据范围 |
| :--- | :--- |
| 销售业务员 | 仅查看本人负责客户的数据（简化版视图） |
| 销售主管 | 查看本团队全部客户的数据 |
| 区域负责人 | 查看本区域全部客户的数据 |
| 系统管理员 | 查看系统全部客户的数据 |


## 十、与其他模块的关系

| 关联模块 | 关系说明 |
| :--- | :--- |
| **客户中心** | 客户经营数据来源于客户中心沉淀的客户档案 |
| **线索中心** | 转化率数据来源于线索转客户的操作记录 |
| **合同中心** | 成交客户数据来源于关联合同的客户去重统计 |
| **工作台** | 工作台展示今日/本周的简要客户指标，客户经营展示周期内的详细分析 |
