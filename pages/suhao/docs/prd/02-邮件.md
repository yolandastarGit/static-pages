## 一、模块概述

**模块归属：** 沟通中心 → 邮件

**模块定位：** 统一管理用户绑定邮箱的所有邮件，提供邮件查看、AI 智能分析、线索生成等核心能力。

**面向角色：** 销售业务员


## 二、页面整体布局

邮件管理页面采用**三栏式布局**：

| 区域 | 宽度 | 内容 |
| :--- | :--- | :--- |
| **左栏（侧边栏）** | 固定 240px | 文件夹导航 |
| **中栏（列表区）** | 自适应，最小 400px | 邮件列表 |
| **右栏（详情区）** | 占比 40% | 邮件详情与 AI 智能分析面板，支持折叠 |


## 三、页面顶部导航

**功能定位：** 邮箱切换与核心操作入口。

| 元素 | 说明 |
| :--- | :--- |
| **邮箱切换器** | 下拉选择框，展示当前用户绑定的所有邮箱。默认选中第一个绑定的邮箱。切换后，列表区刷新为该邮箱的邮件。 |
| **搜索框** | 全局搜索邮件正文、主题、发件人。支持高级筛选（按时间、附件、是否已读）。 |
| **批量操作** | 提供【批量标记已读】、【批量 AI 提炼】。批量 AI 提炼仅限同一邮箱，单次最多可选 50 封。 |


## 四、左栏：文件夹导航

**功能定位：** 邮件分类管理。

| 文件夹 | 说明 | 交互 |
| :--- | :--- | :--- |
| **未读** | 筛选所有未读邮件 | 右上角数字角标提示未读总数 |
| **收件箱** | 收到的邮件 | 默认进入该视图 |
| **已发送** | 已发送的邮件 | — |
| **草稿箱** | 保存中的邮件草稿 | 点击进入草稿列表，支持继续编辑、删除草稿 |
| **垃圾箱** | 逻辑删除的邮件 | 支持恢复 |


## 五、中栏：邮件列表

**功能定位：** 展示邮件核心摘要，快速触达关键信息并进行线索流转。

### 列表字段

| 字段 | 说明 | 展示规则 |
| :--- | :--- | :--- |
| **复选框** | 支持多选，用于触发批量操作 | 位于列表最左侧 |
| **发件人/主题** | 显示发件人姓名和邮件主题 | 未读邮件主题加粗显示 |
| **邮件摘要** | 正文预览，截取前 100 字符 | 超过长度截断加“...” |
| **智能快捷操作** | 置于列表右侧，支持一键标记状态 | 包含【标记待跟进】按钮，点击后为邮件打上“待跟进”标签 |
| **时间** | 显示收件时间 | 今天显示时分（如 10:42），历史日期显示月/日 |

### 列表核心交互

1. **选中态：** 点击列表项，右侧详情面板刷新对应邮件详情。
2. **智能快捷操作：**
   - 【标记待跟进】：点击后为该邮件打上“待跟进”标签，无需进入详情页即可完成意向标注，同时系统自动记录操作日志。
   - 按钮来源：基于 AI 提取的关键词或站点识别结果动态生成。
3. **复选框：** 支持多选，选中后顶部批量操作栏亮起，用于触发批量 AI 提炼等功能。


## 六、右栏：邮件详情与 AI 智能分析面板

**功能定位：** 承载邮件的全部核心数据，是 AI 能力嵌入的核心区域。

### 6.1 邮件基础信息区

展示当前邮件的基础元数据。

| 字段 | 数据来源 | 示例 |
| :--- | :--- | :--- |
| **主题** | 邮件头 | 询盘：定制化工业零件加工 |
| **发件人** | 邮件头 | Elena Rodriguez |
| **时间** | 邮件头 | 2026 年 6 月 28 日 10:42:31 |
| **收件人** | 邮件头 | sales@company.com |
| **来源站点** | 系统匹配 | 官方网站（工业事业部）（若匹配失败，显示为下拉框供手动修正） |

### 6.2 AI 智能分析区

此区域展示 AI 对当前邮件的分析结果。AI 仅作辅助，不阻塞业务。

#### 1）AI 意向智能总结

- **展示内容：** 100-200 字的自然语言总结，包含客户需求、预算信号、痛点及销售建议。
- **交互动作：** 底部提供【复制总结】按钮。

#### 2）AI 企业情报库

- **展示内容：** AI 提取的企业公开信息。
- **字段示例：**
  - 企业名称
  - 行业
  - 年营收（AI 补充信息，仅供销售参考）
  - 员工规模（AI 补充信息，仅供销售参考）
  - 总部所在地（AI 补充信息，仅供销售参考）
- **交互动作：** 若该企业已存在于 CRM，显示【查看已有客户档案】；若不存在，显示【快速创建企业信息库条目】。

> 说明：年营收、员工规模、总部为 AI 补充展示信息，仅供销售参考，不同步写入客户标准字段。

#### 3）AI 风险提醒

- **展示内容：** 安全与合规风险提示。
- **示例：** “发件邮箱与官方域名有细微差异（`@industrial-pro.co` vs `@industrial-pro.com`），建议进行二次身份校验。”

#### 4）AI 标签

- **展示内容：** AI 自动生成的标签，独立于线索的 AI 标签，分别存储。
- **示例：** 高增长潜力、财富 500 强、疑似竞品。

### 6.3 邮件原文区

- **展示形式：** 不可编辑的只读模式，展示邮件完整内容。
- **附件展示：** 展示邮件附件列表，支持点击下载（MVP 支持常见文件类型）。

### 6.4 核心业务操作区

| 操作按钮 | 触发逻辑 |
| :--- | :--- |
| **生成线索（核心）** | 点击后，系统校验该邮件是否已关联线索。<br>• 若未关联：弹出确认生成线索弹窗（预填 AI 企业名称，允许用户修改），确认后执行。<br>• 若已关联：提示“该消息已生成线索 [线索编号]”，点击跳转至线索详情。 |
| **回复** | 跳转至系统默认邮件客户端（或唤起本机邮件客户端），原邮件内容自动带入。 |
| **转发** | 跳转至邮件客户端转发界面。 |


## 七、异常状态与兜底逻辑

### 7.1 AI 服务不可用

- **页面表现：** AI 智能分析区显示灰色占位符，并提示“AI 服务暂时不可用，系统依赖规则引擎正常运行。”
- **业务影响：** 【生成线索】按钮依然可用，后台仅依赖规则识别。

### 7.2 来源站点匹配失败

- **页面表现：** 基础信息区“来源站点”显示为红色高亮下拉框，提示“未能识别来源站点，请手动选择”。
- **业务约束：** 若站点未选择，系统将阻止【生成线索】操作。

### 7.3 邮件加载失败

- **页面表现：** 列表区显示“邮件加载失败，请刷新重试”。
- **交互：** 提供【重新加载】按钮。


## 八、交互说明汇总

| 交互行为 | 说明 |
| :--- | :--- |
| 切换邮箱 | 顶部切换邮箱后，列表和左侧文件夹数据同步刷新 |
| 点击列表项 | 右侧详情面板刷新 |
| 标记待跟进 | 点击列表中的【标记待跟进】按钮，邮件打标，记录操作日志 |
| 批量 AI 提炼 | 勾选多封邮件后点击【批量 AI 提炼】，仅限同一邮箱，上限 50 封 |
| 生成线索 | 弹窗确认，预填 AI 企业名称，用户可修改后提交 |
| 来源站点修正 | 下拉框手动选择，未选择时阻止生成线索 |
| 草稿箱 | 点击进入草稿列表，支持继续编辑和删除草稿 |
