## 一、站点详情页

**触发入口：** 站点列表页 → 点击站点名称

**页面形式：** 独立页面（当前页签打开）

**功能定位：** 只读查看站点的完整信息，包括基础信息、配置摘要、运行状态。

**面向角色：** 系统管理员


### 1.1 页面整体布局

| 区域 | 内容 |
| :--- | :--- |
| **顶部导航区** | 【返回站点列表】按钮 + 站点名称 + 站点编码 + 状态徽章 |
| **统计概览区** | 线索总数 / 客户总数 / 合同总数（该站点下的汇总数据） |
| **信息详情区** | 站点基础信息 + 配置摘要 |


### 1.2 顶部导航区

| 元素 | 说明 |
| :--- | :--- |
| 【返回站点列表】 | 返回站点管理列表页 |
| 站点名称 | 大号字体显示 |
| 站点编码 | 以标签形式展示在名称右侧 |
| 状态徽章 | 启用 / 停用 |


### 1.3 统计概览区

| 统计项 | 说明 |
| :--- | :--- |
| **线索总数** | 该站点下的线索总数量 |
| **客户总数** | 该站点下的客户总数量 |
| **合同总数** | 该站点下的合同总数量 |


### 1.4 信息详情区

| 字段 | 说明 |
| :--- | :--- |
| **站点名称** | 站点名称 |
| **站点编码** | 站点唯一编码 |
| **域名** | 站点官网地址 |
| **联系邮箱** | 站点联系邮箱 |
| **WhatsApp** | 站点 WhatsApp 账号 |
| **创建时间** | 站点创建时间 |
| **状态** | 启用 / 停用 |
| **配置摘要** | 展示该站点的关键配置摘要（业务流程阶段数、公海规则开关、AI规则开关、系统规则开关） |


## 二、站点配置页

**触发入口：** 站点列表页 → 操作列【配置】按钮

**页面形式：** 独立页面（浏览器新开页签打开）

**功能定位：** 编辑站点的全部业务配置参数。

**面向角色：** 系统管理员

**说明：** 配置页新开页签，无返回按钮。配置页顶部保留站点名称 + 站点编码，修改后保存并关闭页签即可。


### 2.1 页面整体布局

| 区域 | 内容 |
| :--- | :--- |
| **顶部信息区** | 站点名称 + 站点编码 + 状态徽章（只读展示，不可编辑） |
| **标签页导航** | 联系方式 / 业务流程 / 公海规则 / AI规则 / 系统规则 |


### 2.2 标签页一：联系方式

**功能定位：** 配置站点的联系邮箱和 WhatsApp 账号。

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| **联系邮箱** | 文本输入 | 否 | 站点联系邮箱，支持多个（分号分隔） |
| **WhatsApp** | 文本输入 | 否 | 站点 WhatsApp 账号 |


### 2.3 标签页二：业务流程

**功能定位：** 配置该站点下的跟进阶段选项列表，不同站点可独立维护。

| 操作 | 说明 |
| :--- | :--- |
| **新增阶段** | 点击新增一行，输入阶段名称 |
| **编辑阶段** | 点击阶段名称行，进入编辑状态 |
| **删除阶段** | 点击删除按钮，二次确认后删除 |
| **阶段排序** | 拖拽调整阶段在列表中的顺序 |

**阶段列表字段：**

| 字段 | 说明 |
| :--- | :--- |
| **阶段名称** | 阶段显示名称（如“首次联系”、“样品阶段”、“需求确认”、“报价”、“合同已成交”等） |
| **触发高意向** | 开关标识，启用表示线索进入该阶段时系统自动将状态变更为“高意向” |
| **排序** | 拖拽手柄调整顺序 |
| **操作** | 编辑 / 删除 |

**校验规则：**
- 阶段名称不可为空
- 同一站点下阶段名称不可重复
- 至少保留一个阶段


### 2.4 标签页三：公海规则

**功能定位：** 配置该站点的公海自动回收规则。

| 配置项 | 组件类型 | 是否必填 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| **启用自动回收** | 开关 | — | 关闭 | 开启后系统定时任务自动执行回收 |
| **回收超时时长** | 数字输入 + 下拉单位 | 是（启用时） | 7 天 | 线索无跟进记录的天数阈值 |


### 2.5 标签页四：AI 规则

**功能定位：** 配置该站点的 AI 识别功能开关。

| 配置项 | 组件类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| **启用 AI 识别** | 开关 | 开启 | 关闭后该站点消息不调用 AI 识别，仅依赖系统规则 |


### 2.6 标签页五：系统规则

**功能定位：** 配置该站点的关键词规则引擎。

**规则列表字段：**

| 字段 | 说明 |
| :--- | :--- |
| **规则名称** | 规则名称 |
| **关键词** | 匹配关键词（如 “buy”、“purchase”、“quote”），多个关键词用逗号分隔 |
| **匹配逻辑** | 包含任一（OR）/ 包含全部（AND） |
| **优先级** | 数字越大优先级越高 |
| **状态** | 启用 / 停用 |
| **操作** | 编辑 / 删除 / 启用/停用 |

**操作按钮：**
- 【新增规则】：弹出新增规则弹窗

**新增/编辑规则弹窗：**

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| 规则名称 | 文本输入 | 是 | 规则名称 |
| 关键词 | 文本输入（标签式） | 是 | 输入关键词后回车添加，支持多个 |
| 匹配逻辑 | 下拉选择 | 是 | 包含任一（OR）/ 包含全部（AND） |
| 优先级 | 数字输入 | 是 | 数字越大优先级越高，默认 0 |
| 状态 | 开关 | 是 | 启用 / 停用 |


## 三、交互说明汇总

| 交互行为 | 说明 |
| :--- | :--- |
| 点击站点名称 | 进入站点详情页（只读查看） |
| 点击配置按钮 | 新开页签进入站点配置页 |
| 配置页保存 | 各标签页独立保存，保存后无需关闭页签，可继续编辑其他标签 |
| 停用站点二次确认 | 列表页点击停用时弹出二次确认弹窗 |


## 四、异常状态与兜底逻辑

| 场景 | 处理方式 |
| :--- | :--- |
| 详情页加载失败 | 显示“数据加载失败，请刷新重试”，提供【重新加载】按钮 |
| 配置页保存失败 | 提示“保存失败，请稍后重试”，保留用户已填内容 |
| 新增阶段名称重复 | 校验拦截，提示“该阶段名称已存在” |
| 删除阶段被线索引用 | 提示“该阶段已被 N 条线索引用，删除后将自动替换为默认阶段，确认删除？” |
| 规则关键词为空 | 校验拦截，提示“请至少添加一个关键词” |


## 五、与其他模块的关系

| 关联模块 | 关系说明 |
| :--- | :--- |
| **线索中心** | 站点配置的阶段选项供线索跟进时选择；系统规则影响线索生成逻辑 |
| **沟通中心** | 站点联系方式用于消息接收和来源识别 |
| **分析中心** | 获客分析中的站点维度数据来源于站点配置 |
| **系统管理** | 站点管理是系统管理员的核心职责之一 |
