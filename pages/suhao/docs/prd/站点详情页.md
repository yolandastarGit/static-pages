## 一、站点详情页

**触发入口：** 站点列表页 → 点击站点名称

**页面形式：** 独立页面（当前页签打开）

**功能定位：** 只读查看站点的完整信息，包括基础信息、运行摘要、运行状态。

**面向角色：** 系统管理员


### 1.1 页面整体布局

| 区域 | 内容 |
| :--- | :--- |
| **顶部导航区** | 【返回站点列表】按钮 + 站点名称 + 站点编码 + 状态徽章 |
| **统计概览区** | 线索总数 / 客户总数 / 合同总数（该站点下的汇总数据） |
| **信息详情区** | 站点基础信息 + 运行摘要 |


### 1.2 顶部导航区

| 元素 | 说明 |
| :--- | :--- |
| 【返回站点列表】 | 返回站点管理列表页 |
| 站点名称 | 大号字体显示 |
| 站点编码 | 以标签形式展示在名称右侧 |
| 状态徽章 | 启用 / 停用 |


### 1.3 统计概览区

| 统计项 | 说明 |
| :--- | :--- |
| **线索总数** | 该站点下的线索总数量 |
| **客户总数** | 该站点下的客户总数量 |
| **合同总数** | 该站点下的合同总数量 |


### 1.4 信息详情区

| 字段 | 说明 |
| :--- | :--- |
| **站点名称** | 站点名称 |
| **站点编码** | 站点唯一编码 |
| **域名** | 站点官网地址 |
| **联系邮箱** | 站点联系邮箱 |
| **WhatsApp** | 站点 WhatsApp 账号 |
| **创建时间** | 站点创建时间 |
| **状态** | 启用 / 停用 |
| **运行摘要** | 展示该站点的关键运行摘要（业务流程阶段数、公海规则开关、AI规则开关、系统规则开关） |


## 二、站点运行信息

**触发入口：** 站点列表页 → 点击站点名称

**页面形式：** 站点详情页内展示

**功能定位：** 展示单个站点的消息拉取、接口状态及业务摘要，不提供独立配置入口。

**面向角色：** 系统管理员

### 2.1 页面整体布局

| 区域 | 内容 |
| :--- | :--- |
| **指标区** | 线索总数 / 客户总数 / 合同总数 / 接口状态 |
| **基础信息区** | 站点名称、编码、域名、联系方式、创建时间 |
| **运行摘要区** | 阶段数量、公海状态、AI 状态、系统规则状态 |


## 三、交互说明汇总

| 交互行为 | 说明 |
| :--- | :--- |
| 点击站点名称 | 进入站点详情页（只读查看） |
| 停用站点二次确认 | 列表页点击停用时弹出二次确认弹窗 |


## 四、异常状态与兜底逻辑

| 场景 | 处理方式 |
| :--- | :--- |
| 详情页加载失败 | 显示“数据加载失败，请刷新重试”，提供【重新加载】按钮 |
| 站点不存在 | 显示“站点不存在，请返回站点管理重新选择” |


## 五、与其他模块的关系

| 关联模块 | 关系说明 |
| :--- | :--- |
| **线索中心** | 站点作为线索来源归属，支持按站点查看相关线索 |
| **沟通中心** | 站点联系方式用于消息接收和来源识别 |
| **分析中心** | 获客分析中的站点维度数据来源于站点基础数据 |
| **系统管理** | 站点管理是系统管理员的核心职责之一 |
