## 一、模块概述

**模块归属：** 分析中心 → 销售经营

**模块定位：** 提供销售核心指标的经营分析，帮助管理者了解线索总量、成交金额、转化效率、销售漏斗及团队排行。

**面向角色：** 销售主管、区域负责人


## 二、页面整体布局

销售经营页面采用 **顶部筛选栏 + 指标卡区 + 图表区 + 排行榜区** 布局：

| 区域 | 内容 |
| :--- | :--- |
| **顶部筛选栏** | 时间快捷选项 + 站点筛选（仅管理员/区域负责人可见） |
| **指标卡区** | 4 个核心指标卡片 + 辅助指标（较上期变化量） |
| **图表区（左）** | 销售漏斗分析 |
| **图表区（右）** | 线索增长与成交金额趋势（折线图） |
| **排行榜区** | 销售排行榜 TOP10 |


## 三、顶部筛选栏

**功能定位：** 统一控制页面所有图表的数据范围。

| 筛选项 | 类型 | 说明 | 默认值 |
| :--- | :--- | :--- | :--- |
| **时间快捷选项** | 按钮组 | 今日 / 本周 / 本月 / 本季度 / 自定义 | 本月 |
| **站点筛选** | 下拉选择 | 按站点过滤数据，仅管理员/区域负责人可见 | 全部站点 |

**交互规则：**
- 点击时间快捷选项，页面所有图表数据同步刷新
- 自定义时间支持选择起始日期和结束日期
- 站点筛选切换后，所有图表按站点过滤


## 四、指标卡区

位于筛选栏下方，一行展示 4 个核心指标卡片。

### 卡片一：线索总数

**定义：** 截至当前时间，系统中的线索总数量。

| 元素 | 说明 |
| :--- | :--- |
| 主数值 | 线索总数（如 12,482） |
| 变化量 | 较上期变化量，格式：较上期增加/减少 X，XX（如“较上期增加 1,540”） |
| 目标达成 | 今日目标达成率，格式：今日目标达成率 XX%（如“今日目标达成率 88%”） |

### 卡片二：新增线索

**定义：** 在当前统计周期内（默认本月），新创建的线索数量。

| 元素 | 说明 |
| :--- | :--- |
| 主数值 | 新增线索数（如 842） |
| 变化量 | 较上期变化量，格式：~+/-X.X%（如“~+4.2%”） |
| 目标达成 | 年度目标完成率，格式：年度目标完成 XX%（如“年度目标完成 62%”） |

### 卡片三：成交金额

**定义：** 在当前统计周期内（默认本月），已签订合同的金额累计总和。

| 元素 | 说明 |
| :--- | :--- |
| 主数值 | 累计成交金额（含币种，如 ¥4.2M） |
| 变化量 | 较上期变化量，格式：~+/-X.X%（如“~+2.1%） |

### 卡片四：线索转化率

**定义：** 在当前所选统计周期内，已转化为客户的线索数占有效线索数的百分比。

**计算公式：** `线索转化率 = 所选周期内已成交线索数 / 所选周期内有效线索数 × 100%`

| 元素 | 说明 |
| :--- | :--- |
| 主数值 | 转化率百分比（如 24.8%） |
| 变化量 | 较上期变化量，格式：~+/-X.X%（如“~+0.5%） |
| 基准对比 | 行业平均水平（如“行业平均水平 18%”） |


## 五、销售漏斗分析

**图表类型：** 横向漏斗图

**功能定位：** 展示线索从消息进入到最终转化的各阶段数量与转化率，帮助识别转化瓶颈。

### 漏斗阶段（从左到右）：

| 阶段 | 说明 | 示例数量 | 转化率 |
| :--- | :--- | :--- | :--- |
| **全部消息** | 选定时间范围内进入系统的全部消息数（不去重） | 25,000 | 100% |
| **识别为线索** | 选定时间范围内经识别确认为有效并生成线索的数量 | 12,482 | 49.9% |
| **跟进中** | 选定时间范围内已分配并进入跟进阶段的线索数（含已成交、已关闭） | 8,240 | 32.9% |
| **高意向** | 选定时间范围内达到高意向阶段的线索数 | 3,120 | 12.5% |
| **转客户** | 选定时间范围内最终转化为客户的数量 | 1,220 | 4.9% |

### 展示规则：

| 元素 | 说明 |
| :--- | :--- |
| 图表类型 | 横向漏斗图（左宽右窄） |
| 数值展示 | 每个阶段显示绝对数量 |
| 转化率 | 相邻阶段之间显示转化百分比 |
| 颜色 | 各阶段使用渐变色，从深到浅 |


## 六、线索增长与成交金额趋势

**图表类型：** 双轴折线图/柱线组合图

**功能定位：** 展示线索数量与成交金额的月度变化趋势，帮助分析销售节奏。

### 数据维度：

| 数据线 | 说明 | Y 轴 |
| :--- | :--- | :--- |
| **线索数量** | 每月新增线索数 | 左 Y 轴（数量） |
| **成交金额** | 每月合同成交金额 | 右 Y 轴（金额） |

### 展示规则：

- X 轴：月份（本年度 1 月至当前月）
- 左 Y 轴：线索数量（柱状图）
- 右 Y 轴：成交金额（折线图）
- 数据点标注具体数值
- 空态处理：无数据时显示“暂无数据”


## 七、销售排行榜

**功能定位：** 展示销售人员的业绩排行，激励团队竞争。

### 顶部 Tab 切换：

| Tab | 排序依据 |
| :--- | :--- |
| **按成交额** | 按成交金额降序排列 |
| **按转客数** | 按转客户数降序排列 |

### 表格字段：

| 字段 | 说明 |
| :--- | :--- |
| **排名** | 1 ~ 10 |
| **业务员** | 姓名 + 所属团队 |
| **线索数量** | 该业务员负责的线索总数 |
| **跟进数** | 该业务员的跟进记录总数 |
| **转客户数** | 该业务员将线索转为客户的数量 |
| **成交金额** | 该业务员负责的合同累计金额 |
| **目标达成** | 实际成交金额 / 个人目标金额 × 100% |

### 展示规则：

- 默认展示 TOP10
- 按成交额 Tab 时，按成交金额降序排列
- 按转客数 Tab 时，按转客户数降序排列
- 当前用户本人所在行高亮显示
- 分页：每页 10 条

## 九、空态与异常态

| 场景 | 处理方式 |
| :--- | :--- |
| 无任何数据 | 指标卡主数值显示"--"，图表显示空状态占位图 |
| 漏斗图无数据 | 显示空漏斗轮廓，每阶段标注"--" |
| 趋势图无数据 | 坐标系保留（X/Y 轴），折线区域显示“暂无趋势数据” |
| 排行榜无数据 | 列表区域显示“暂无排行数据” |
| 部分月份无数据 | 趋势图中该月份显示为 0 或断点 |
| 数据加载中 | 图表区域显示骨架屏 |
| 数据加载失败 | 显示“数据加载失败，请刷新重试”，提供【重新加载】按钮 |

## 十二、目标金额维护入口

### 12.1 数据来源

销售经营页面中"目标金额"数据来源于 [系统管理 → 参数设置](file:///Users/demo-user/Documents/项目-苏豪 0.2/prd/013-系统管理.md) → **个人目标金额** 参数项。

### 12.2 配置项

| 参数键 | 默认值 | 生效时间 | 维护权限 |
| :--- | :--- | :--- | :--- |
| 个人目标金额 | 0（无目标） | 立即生效 | 系统管理员 |

### 12.3 应用规则

- 单个业务员的目标 = 系统级"个人目标金额"参数值
- 团队/区域目标 = 团队/区域成员"个人目标金额"之和
- 当参数值为 0 时，目标达成度显示为"--"，不参与达成率计算
- 修改参数后，本页面的"目标金额"和"目标达成"指标实时刷新

参数设置的详细字段定义：参见 [013-系统管理.md](file:///Users/demo-user/Documents/项目-苏豪 0.2/prd/013-系统管理.md) § 六 参数设置 → 个人目标金额
