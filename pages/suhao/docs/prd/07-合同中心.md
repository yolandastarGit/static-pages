## 一、模块概述

**模块归属：** 客户中心 → 合同中心

**模块定位：** 统一管理所有销售合同，记录客户成交信息，支撑经营分析与成交追溯。

**面向角色：** 销售业务员、销售主管


## 二、页面整体布局

合同中心页面采用 **单页表格布局**，从上至下分为两个区域：

| 区域 | 内容 |
| :--- | :--- |
| **筛选条件栏** | 多维度筛选条件，支持精细化查询 |
| **表格区** | 合同数据表格，含操作按钮及分页 |


## 三、筛选条件栏

**功能定位：** 提供多维度筛选条件，支持精细化查询。

| 筛选项 | 类型 | 说明 | 默认值 |
| :--- | :--- | :--- | :--- |
| **合同编号** | 文本输入 | 模糊搜索合同编号 | 空 |
| **客户名称** | 文本输入 | 模糊搜索客户名称或客户邮箱 | 空 |
| **签约日期** | 日期区间 | 按合同签约日期范围筛选 | 本月 |
| **合同状态** | 下拉选择 | 按合同状态筛选 | 全部 |
| **负责人** | 下拉选择 | 按合同负责人筛选（关联线索的负责人） | 全部 |

**操作按钮：**

| 按钮 | 说明 |
| :--- | :--- |
| **查询** | 按当前筛选条件刷新表格数据 |
| **重置** | 将所有筛选条件恢复为默认值 |
| **导出合同** | 将当前筛选结果导出为 CSV 文件，受操作权限控制 |
| **高级筛选** | 展开更多筛选条件 |


## 四、表格区

### 4.1 顶部操作按钮

位于表格左上方，提供合同操作入口：

| 按钮 | 说明 | 前置条件 |
| :--- | :--- | :--- |
| **新增合同** | 弹出新增合同弹窗，录入新合同 | 无条件 |
| **导入合同** | 批量导入合同（功能预留，本期不启用） | — |

### 4.2 表格字段

| 字段 | 说明 | 展示规则 |
| :--- | :--- | :--- |
| **合同编号** | 合同唯一编号 | 固定格式展示，如 PC-2026-0120，点击可跳转合同详情页 |
| **合同名称** | 合同名称 | 展示合同名称 |
| **客户** | 关联的客户名称 | 展示客户名称 |
| **金额** | 合同金额 | 含币种符号，如 $9,800 / ¥45,200 |
| **签约日期** | 合同签约日期 | 展示日期（YYYY-MM-DD） |
| **合同状态** | 合同当前状态 | 已签约 / 执行中 / 已完成 / 已终止 / 已作废，以彩色徽章展示 |
| **负责人** | 关联线索的负责人 | 展示负责人姓名（继承自关联线索） |
| **操作** | 单条合同操作入口 | 包含：【详情】、【编辑金额】、【编辑】、【更多】下拉 |


## 五、新增合同弹窗

**功能定位：** 录入新合同，记录客户成交信息。

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| **合同编号** | 文本输入 | 是 | 人工录入，系统不自动生成，格式不做强制校验 |
| **合同名称** | 文本输入 | 是 | 合同名称 |
| **客户** | 下拉选择（搜索选择） | 是 | 可选当前用户有权限查看的客户列表，支持按客户名称搜索 |
| **关联线索** | 下拉选择（搜索选择） | 否 | 可选该客户下的线索列表，非必填；选填后用于成交来源追溯 |
| **金额** | 数字输入 | 是 | 合同金额，仅支持数字 |
| **币种** | 下拉选择 | 是 | 默认：美元（USD），选项：美元 / 人民币 / 欧元 / 英镑等 |
| **签约日期** | 日期选择器 | 是 | 默认：今日（用户可手动修改） |
| **合同状态** | 下拉选择 | 是 | 默认：已签约（用户可手动修改） |

**关联线索下拉选项逻辑：**
- 仅展示已选客户下的线索
- 未选择客户时，关联线索下拉不可用（置灰）
- 线索状态不限制，所有状态的线索均可关联

**提交校验：**

| 校验项 | 校验规则 | 错误提示 |
| :--- | :--- | :--- |
| 合同编号 | 不可为空 | “请输入合同编号” |
| 合同名称 | 不可为空 | “请输入合同名称” |
| 客户 | 不可为空 | “请选择客户” |
| 金额 | 不可为空，且必须为大于 0 的数字 | “请输入有效金额” |
| 币种 | 不可为空 | “请选择币种” |

**提交逻辑：**
1. 创建合同记录，状态按用户选择写入
2. 合同编号由用户录入，系统不自动生成
3. 若关联线索已选，建立合同与线索的关联关系
4. 合同“负责人”取关联线索的负责人；若未关联线索，负责人为空，允许后续编辑补充
5. 自动生成跟进记录（跟进方式：备注，内容：“录入合同 [合同编号]”）

**合同编号唯一性校验：**
- 当前站点下，合同编号不可重复
- 重复时提示“该合同编号已存在，请重新输入”


## 六、操作列详细说明

| 操作 | 说明 | 权限 |
| :--- | :--- | :--- |
| **详情** | 跳转至合同详情页 | 无条件 |
| **编辑金额** | 弹出编辑弹窗，仅允许修改**金额**字段，币种不可编辑 | 当前合同负责人本人或销售主管 |
| **编辑** | 弹出编辑弹窗，允许修改**合同名称、签约日期、合同状态** | 当前合同负责人本人或销售主管 |
| **更多** | 下拉菜单，包含【作废】、【终止】 | 当前合同负责人本人或销售主管 |

### 6.1 编辑金额

| 字段 | 组件类型 | 是否可编辑 | 说明 |
| :--- | :--- | :--- | :--- |
| 合同编号 | 只读展示 | 否 | — |
| 合同名称 | 只读展示 | 否 | — |
| 客户 | 只读展示 | 否 | — |
| 金额 | 数字输入 | 是 | 仅允许修改金额数值 |
| 币种 | 只读展示 | 否 | 币种不可变更 |

提交成功后自动生成跟进记录（跟进方式：备注，内容：“合同金额由 [原值] 变更为 [新值]”）

### 6.2 编辑

| 字段 | 组件类型 | 是否可编辑 | 说明 |
| :--- | :--- | :--- | :--- |
| 合同编号 | 只读展示 | 否 | — |
| 合同名称 | 文本输入 | 是 | 修改合同名称 |
| 客户 | 只读展示 | 否 | 客户不可变更 |
| 签约日期 | 日期选择器 | 是 | 修改签约日期 |
| 合同状态 | 下拉选择 | 是 | 修改合同状态 |
| 负责人 | 只读展示 | 否 | 负责人不可在此编辑 |

提交成功后自动生成跟进记录（跟进方式：备注，内容：“合同信息已更新”）

### 6.3 更多下拉

| 操作 | 说明 | 状态限制 |
| :--- | :--- | :--- |
| **作废** | 将合同状态变更为“已作废”，二次确认后执行 | 仅当前状态为“已签约”或“执行中” |
| **终止** | 将合同状态变更为“已终止”，二次确认后执行 | 仅当前状态为“执行中” |


## 七、合同详情页

**触发入口：** 点击操作列【详情】按钮或点击表格中【合同编号】

**页面形式：** 独立页面

### 7.1 页面内容

| 区域 | 内容 |
| :--- | :--- |
| **顶部信息区** | 合同编号、合同名称、合同状态（徽章）、负责人 |
| **基本信息区** | 客户名称（点击跳转客户详情）、金额、币种、签约日期 |
| **关联信息区** | 关联线索（展示线索编号及名称，点击跳转线索详情） |
| **操作记录区** | 该合同相关的跟进记录列表（录入、编辑金额、编辑、作废、终止等操作均生成记录） |


## 八、合同状态流转规则

| 当前状态 | 允许流转至 | 触发操作 |
| :--- | :--- | :--- |
| 已签约 | 执行中 | 手动编辑状态 |
| 已签约 | 已作废 | 更多→作废 |
| 执行中 | 已完成 | 手动编辑状态 |
| 执行中 | 已终止 | 更多→终止 |
| 执行中 | 已作废 | 更多→作废 |
| 已完成 | — | 终态，不可流转 |
| 已终止 | — | 终态，不可流转 |
| 已作废 | — | 终态，不可流转 |


## 九、异常状态与兜底逻辑

| 场景 | 处理方式 |
| :--- | :--- |
| 合同列表无数据 | 显示“暂无合同”占位图 |
| 搜索无结果 | 显示“未找到匹配的合同”，建议“请调整筛选条件后重试” |
| 数据加载失败 | 显示“数据加载失败，请刷新重试”，提供【重新加载】按钮 |
| 合同编号重复 | 校验拦截，提示“该合同编号已存在，请重新输入” |
| 已作废/已终止合同不可编辑 | 操作列中编辑金额、编辑、更多（作废/终止）按钮置灰或隐藏 |
| 导入合同（预留功能） | 本期不启用，按钮隐藏或置灰 |

| 点击查询 | 按当前筛选条件刷新表格 |
| 点击重置 | 所有筛选条件恢复默认值 |
| 点击新增合同 | 弹出新增合同弹窗 |

## 十一、与其他模块的关系

| 关联模块 | 关系说明 |
| :--- | :--- |
| **客户中心** | 合同关联客户，新增合同时必须选择客户；客户详情页展示关联合同列表 |
| **线索中心** | 合同关联线索（非必填），用于成交来源追溯；关联线索的负责人自动成为合同负责人 |
| **线索列表** | 线索详情页展示关联合同列表 |
| **跟进记录** | 新增合同、编辑金额、编辑、作废、终止等操作均自动生成跟进记录 |
