## 一、模块概述

**模块归属：** 系统管理

**模块定位：** 提供平台级配置与管理能力，包含用户管理、AI能力配置、业务规则配置、消息提醒规则、系统日志、系统参数。角色权限作为用户管理下的角色属性配置能力保留，不再作为独立二级菜单展示。

**面向角色：** 系统管理员

菜单结构
├── 系统管理
│   ├── 用户管理
│   ├── 系统日志
│   └── 系统参数

## 二、用户管理

### 2.1 功能定位

管理后台所有用户账号，包括新增、编辑、启用/停用、重置密码、分配角色、分配站点。

### 2.2 页面布局

用户管理页面采用 **单页表格布局**：

| 区域 | 内容 |
| :--- | :--- |
| **顶部操作栏** | 搜索框 + 【新增用户】按钮 |
| **表格区** | 用户列表 + 分页 |

### 2.3 表格字段

| 字段 | 说明 |
| :--- | :--- |
| **姓名** | 用户姓名 |
| **登录账号** | 系统登录账号（唯一） |
| **手机号** | 用户手机号 |
| **邮箱** | 用户邮箱 |
| **角色** | 所属角色 |
| **所属站点** | 授权的站点 |
| **状态** | 启用 / 停用 |
| **操作** | 编辑 / 启用/停用 / 重置密码 |

**角色权限入口：**
- 用户管理页面提供【角色权限配置】入口
- 点击后进入角色权限页面，配置角色的菜单权限、按钮权限、数据权限
- 角色权限页面路由、权限控制和业务逻辑保持不变


### 2.4 新增/编辑用户弹窗

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| **姓名** | 文本输入 | 是 | 用户姓名 |
| **登录账号** | 文本输入 | 是 | 唯一，不可重复 |
| **手机号** | 文本输入 | 否 | 用户手机号 |
| **邮箱** | 文本输入 | 否 | 用户邮箱 |
| **角色** | 下拉选择 | 是 | 从角色列表中选择 |
| **所属站点** | 下拉选择（多选） | 是 | 可授权多个站点 |
| **状态** | 开关 | 是 | 启用 / 停用 |

**操作说明：**
- 新增用户时，系统自动生成初始密码（默认 123456），用户首次登录强制修改
- 编辑用户时，不支持修改登录账号
- 重置密码：将用户密码重置为初始密码（123456）


## 三、角色权限（用户管理下的功能入口）

### 3.1 功能定位

管理角色列表，并为角色配置菜单权限、按钮权限、数据权限。

### 3.2 页面布局

角色权限页面采用 **左侧角色列表 + 右侧权限配置** 布局：

| 区域 | 宽度 | 内容 |
| :--- | :--- | :--- |
| **左栏（角色列表）** | 240px | 角色列表 + 新增/删除角色 |
| **右栏（权限配置）** | 自适应 | 菜单权限、按钮权限、数据权限配置 |

### 3.3 权限配置内容

| 权限类型 | 说明 | 配置方式 |
| :--- | :--- | :--- |
| **菜单权限** | 控制左侧导航菜单可见性 | 树形勾选（全选/取消全选） |
| **按钮权限** | 控制页面内操作按钮可见性 | 按模块分组勾选 |
| **数据权限** | 控制数据行级可见范围 | 下拉选择：仅本人 / 本团队 / 本区域 / 全部 |

## 四、系统日志

### 4.1 功能定位

查看系统登录日志和操作日志，用于安全审计与问题追溯。

### 4.2 页面布局

系统日志页面采用 **标签页切换 + 表格展示** 布局：

| 标签页 | 内容 |
| :--- | :--- |
| **登录日志** | 记录所有用户登录行为 |
| **操作日志** | 记录所有业务操作行为 |

### 4.3 登录日志

**筛选条件：**

| 筛选项 | 类型 | 说明 | 默认值 |
| :--- | :--- | :--- | :--- |
| 时间范围 | 日期区间 | 按登录时间筛选 | 最近7天 |
| 登录账号 | 文本输入 | 模糊搜索 | 空 |
| 登录结果 | 下拉选择 | 成功 / 失败 | 全部 |

**表格字段：**

| 字段 | 说明 |
| :--- | :--- |
| 登录时间 | 精确到秒 |
| 登录账号 | 系统账号 |
| 用户姓名 | 用户姓名 |
| 登录 IP | 客户端 IP 地址 |
| 登录方式 | 账号密码 / 钉钉扫码 |
| 登录结果 | 成功 / 失败 |
| 浏览器 | User-Agent 解析 |
| 操作系统 | User-Agent 解析 |

### 4.4 操作日志

**筛选条件：**

| 筛选项 | 类型 | 说明 | 默认值 |
| :--- | :--- | :--- | :--- |
| 时间范围 | 日期区间 | 按操作时间筛选 | 最近7天 |
| 操作人 | 文本输入 | 模糊搜索 | 空 |
| 操作类型 | 下拉选择 | 新增 / 编辑 / 删除 / 导出 / 登录 / 配置 | 全部 |
| 业务对象 | 下拉选择 | 用户 / 角色 / 站点 / 线索 / 客户 / 合同 | 全部 |

**表格字段：**

| 字段 | 说明 |
| :--- | :--- |
| 操作时间 | 精确到秒 |
| 操作人 | 操作人姓名 |
| 操作类型 | 新增 / 编辑 / 删除 / 导出 / 登录 / 配置 |
| 业务对象 | 操作的目标对象类型 |
| 对象名称 | 操作的目标对象名称 |
| 操作内容 | 操作详情描述 |
| IP 地址 | 操作人 IP |


## 五、系统参数
### 5.1 AI 能力配置
### 5.11 功能定位

配置全局 AI 服务商、模型与 API 参数，统一管理 AI 能力。站点级 AI 规则仅控制该站点的 AI 识别开关（启用/停用），全局配置与站点级配置独立。

### 5.12 配置字段

| 配置项 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| **AI 服务商** | 下拉选择 | 是 | OpenAI / Azure / 自定义 |
| **模型** | 下拉选择 | 是 | gpt-4o / gpt-4-turbo / 其他 |
| **API Key** | 密码输入 | 是 | API 密钥 |
| **API Endpoint** | 文本输入 | 是 | API 地址 |
| **启用 AI 增强能力** | 开关 | 否 | 全局启用/停用 AI 能力 |
| **本月用量** | 只读展示 | — | 已使用次数 / 总配额次数 |

### 5.13 操作按钮

| 按钮 | 说明 |
| :--- | :--- |
| **测试连接并启用** | 测试当前配置是否可用，测试通过后启用 |
| **保存配置** | 保存当前配置，不执行测试 |
| **关闭 AI 能力** | 停用全局 AI 能力 |

### 5.14 配置信息展示

| 信息项 | 说明 |
| :--- | :--- |
| **最近配置** | 最近一次配置更新时间、服务商、模型 |
| **最近测试** | 最近一次测试时间、测试结果 |
| **审计记录** | AI 生成内容记录：生成时间、使用模型、操作人/模块来源 |


## 5.2、业务规则配置

### 5.21 功能定位

配置系统级业务规则，包括线索入池、邮件采集、线索过期、客户分配、合同归属等规则。

### 5.22 规则列表

**线索入池规则**

| 规则名称 | 规则说明 | 当前配置 | 操作 |
| :--- | :--- | :--- | :--- |
| 有效询盘自动入池 | 站点接口、网站表单等渠道识别的有效询盘，自动进入公海池等待分配。 | 开启 | 编辑 |

**邮件采集规则**

| 规则名称 | 规则说明 | 当前配置 | 操作 |
| :--- | :--- | :--- | :--- |
| 采集邮箱来件自动创建线索 | 运营邮箱收到来件后，按解析规则生成线索或进入待确认队列。 | 开启 | 编辑 |
| 陌生发件人自动创建线索 | 发件人未匹配现有客户/联系人时，是否自动创建新线索。 | 关闭 | 编辑 |

**线索过期规则**

| 规则名称 | 规则说明 | 当前配置 | 操作 |
| :--- | :--- | :--- | :--- |
| 未跟进自动回收天数 | 业务员负责线索在期限内无任何跟进记录时，系统自动回收至公海池。 | 14 天 | 编辑 |

> 说明：未跟进自动回收天数同时受站点级配置控制，以站点配置为准。

**客户分配规则**

| 规则名称 | 规则说明 | 当前配置 | 操作 |
| :--- | :--- | :--- | :--- |
| 客户负责人变更是否同步线索负责人 | 客户转移或负责人调整时，关联线索负责人是否一并更新。 | 是 | 编辑 |

**合同归属规则**

| 规则名称 | 规则说明 | 当前配置 | 操作 |
| :--- | :--- | :--- | :--- |
| 新增合同自动归属客户负责人 | 录入或关联合同时，默认将合同业务员设为客户当前负责人。 | 是 | 编辑 |

### 5.3 规则编辑弹窗

| 字段 | 组件类型 | 说明 |
| :--- | :--- | :--- |
| 规则名称 | 只读展示 | 规则名称 |
| 规则说明 | 只读展示 | 规则说明 |
| 当前配置 | 开关 / 数字输入 / 下拉选择 | 根据规则类型动态变化 |

## 九、与其他模块的关系

| 关联模块 | 关系说明 |
| :--- | :--- |
| **站点中心** | 用户管理中分配站点；AI能力配置为全局配置，站点级AI规则仅控制开关 |
| **登录模块** | 用户管理和角色权限控制登录后的权限加载 |
| **线索中心** | 业务规则配置影响线索生成、入池、回收逻辑 |
| **合同中心** | 合同归属规则影响合同创建时的负责人默认值 |
| **分析中心** | 系统参数不直接影响分析中心，但影响整体系统运行 |
