## 一、模块概述

**模块归属：** 个人中心

**模块定位：** 维护当前登录用户的个人信息、账号绑定、密码修改及安全设置。

**面向角色：** 全部角色


## 二、页面整体布局

个人中心页面采用 **卡片平铺布局**，所有设置项在同一页面内从上至下依次排列：

| 区域 | 内容 |
| :--- | :--- |
| **我的资料** | 维护当前用户的基本个人信息（头像、姓名、手机号、邮箱） |
| **账号绑定** | 管理邮箱绑定和 WhatsApp 绑定 |
| **修改密码** | 修改当前用户的登录密码 |
| 各区域以独立卡片形式展示，无需侧边导航切换。


## 三、我的资料

**功能定位：** 维护当前用户的基本个人信息。

### 3.1 展示与编辑字段

| 字段 | 组件类型 | 是否可编辑 | 说明 |
| :--- | :--- | :--- | :--- |
| **头像** | 图片上传 | 是 | 支持上传图片，建议尺寸 200x200px，支持 jpg/png 格式 |
| **姓名** | 文本输入 | 是 | 用户姓名 |
| **手机号** | 文本输入 | 是 | 用户手机号 |
| **邮箱** | 文本输入 | 是 | 用户邮箱 |

### 3.2 交互规则

- 点击【保存】按钮提交修改
- 所有字段均为可选，用户可仅修改部分字段
- 保存成功后提示“个人信息已更新”


## 四、账号绑定

**功能定位：** 绑定邮箱和 WhatsApp 账号，用于接收消息和自动识别来源。

### 4.1 邮箱绑定

**功能说明：** 支持绑定多个邮箱账号，系统统一归集至当前用户。

| 操作 | 说明 |
| :--- | :--- |
| **新增绑定** | 点击【新增绑定】按钮，弹出绑定弹窗，输入邮箱地址后确认 |
| **删除绑定** | 点击邮箱行【删除】按钮，二次确认后解除绑定 |
| **设置默认** | 不提供设置默认邮箱功能，以首个绑定邮箱为默认 |

**绑定邮箱列表：**

| 字段 | 说明 |
| :--- | :--- |
| 邮箱地址 | 绑定的邮箱账号 |
| 绑定时间 | 绑定时间 |
| 操作 | 删除 |

**新增绑定弹窗：**

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| 邮箱地址 | 文本输入 | 是 | 输入邮箱地址，校验格式合法性 |

**校验规则：**
- 邮箱地址不可为空
- 邮箱地址格式必须合法
- 不可重复绑定同一邮箱
- 同一邮箱不可被多个用户绑定

### 4.2 WhatsApp 绑定

**功能说明：** MVP 仅支持绑定一个 WhatsApp 账号。

| 操作 | 说明 |
| :--- | :--- |
| **绑定** | 点击【绑定】按钮，弹出绑定弹窗，输入 WhatsApp 账号后确认 |
| **解绑** | 点击【解绑】按钮，二次确认后解绑，解绑后可重新绑定新的 WhatsApp |

**校验规则：**
- WhatsApp 账号不可为空
- 一个 WhatsApp 账号不可被多个用户绑定


## 五、修改密码

**功能定位：** 修改当前用户的登录密码。

### 5.1 表单字段

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| **新密码** | 密码输入 | 是 | 新密码，支持显示/隐藏 |
| **确认新密码** | 密码输入 | 是 | 再次输入新密码确认 |

### 5.2 校验规则

| 校验项 | 规则 | 错误提示 |
| :--- | :--- | :--- |
| 新密码 | 不可为空 | “请输入新密码” |
| 新密码 | 长度不少于 8 位 | “密码长度不少于 8 位” |
| 新密码 | 符合密码复杂度规则 | “密码需包含数字和大小写字母” |
| 确认新密码 | 不可为空 | “请再次输入新密码” |
| 确认新密码 | 与新密码一致 | “两次输入的密码不一致” |

### 5.3 交互规则

- 点击【确认修改】按钮提交
- 修改成功后提示“密码已修改，请重新登录”
- 系统自动退出登录，跳转至登录页


## 六、交互说明汇总

| 交互行为 | 说明 |
| :--- | :--- |
| 滚动页面 | 各卡片区域按顺序展示，无需切换 |
| 保存我的资料 | 提交修改，成功后提示 |
| 新增邮箱绑定 | 弹出弹窗，输入邮箱，校验后绑定 |
| 删除邮箱绑定 | 二次确认后解绑 |
| 绑定 WhatsApp | 弹出弹窗，输入账号，校验后绑定 |
| 解绑 WhatsApp | 二次确认后解绑 |
| 修改密码 | 提交后校验，成功后退出登录 |
| 退出登录 | 清除会话，跳转至登录页 |


## 七、异常状态与兜底逻辑

| 场景 | 处理方式 |
| :--- | :--- |
| 头像上传格式不支持 | 提示“请上传 jpg 或 png 格式图片” |
| 头像上传大小超限 | 建议限制 2MB，提示“图片大小不能超过 2MB” |
| 邮箱已被其他用户绑定 | 提示“该邮箱已被其他用户绑定” |
| WhatsApp 已被其他用户绑定 | 提示“该 WhatsApp 已被其他用户绑定” |
| 新密码与确认密码不一致 | 提交时校验拦截，提示“两次输入的密码不一致” |


## 八、与其他模块的关系

| 关联模块 | 关系说明 |
| :--- | :--- |
| **登录模块** | 修改密码后自动退出登录，需重新登录 |
| **沟通中心** | 账号绑定的邮箱和 WhatsApp 用于消息接收和来源识别 |
| **用户管理** | 个人中心维护用户个人信息，与管理员在用户管理中维护的信息同步 |
