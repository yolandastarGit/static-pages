# 菜单结构

## 一、菜单树总览

```
├── 工作台
│
├── 沟通中心
│   ├── 邮件
│   └── WhatsApp
│
├── 线索中心
│   ├── 线索列表
│   └── 公海池
│
├── 客户中心
│   ├── 客户列表
│   └── 合同中心
│
├── 分析中心
│   ├── 销售经营
│   ├── 获客分析
│   └── 客户经营
│
├── 站点中心
│   ├── 站点管理
│   └── 站点运营数据
│
├── AI能力管理
│
├── 通知管理
│   └── 通知场景
└── 系统管理
    ├── 用户管理
    ├── 角色管理
    ├── 菜单管理
    ├── 字典管理
    ├── 系统参数
    ├── 集成配置（邮箱 / 钉钉 / 文件存储 / Apollo 客户画像）
    └── 系统日志
```

## 二、不占侧边菜单的入口

| 入口 | 路径 | 备注 |
| :--- | :--- | :--- |
| 登录页 | /login | 独立路由 |
| 线索详情 | 工作区页签 | 新开页签，不做抽屉；不占侧边菜单（对齐 PRD §11） |
| 通知收件（站内信） | Header 头像左侧铃铛 | 查看本人通知内容；无独立承接页面（对齐 PRD §22.5） |
| 个人中心 | Header 头像下拉 | MVP：弹窗承载我的资料/账号绑定/修改密码；邮箱绑定含密码/授权码，运营须选关联站点（对齐 PRD §24.3.1） |
| 状态页 | 系统级 | 404/403/500/维护中/网络异常/Loading |
| 全局导航 | 系统框架 | 顶部栏/侧边栏/面包屑 |
| 全局搜索 | Header | 【二期/延后】MVP 不提供入口 |

## 三、个人中心入口

个人中心不设侧边菜单。**MVP：** 点击右上角头像下拉，分别打开「我的资料 / 账号绑定 / 修改密码」弹窗（资料只读）。**【二期/延后】** 再改为进入独立页面后以卡片/列表平铺展示全部设置项。
