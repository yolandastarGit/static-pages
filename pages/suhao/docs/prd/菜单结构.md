# 菜单结构

## 一、菜单树总览

```
├── 工作台
│
├── 沟通中心
│   ├── 邮件
│   └── WhatsApp
│
├── 线索中心
│   ├── 线索列表
│   └── 公海池
│
├── 客户中心
│   ├── 客户列表
│   └── 合同中心
│
├── 分析中心
│   ├── 销售经营
│   ├── 获客分析
│   └── 客户经营
│
├── 站点中心
│   └── 站点管理
│
├── AI能力管理
│
├── 通知管理
│   └── 通知中心
│
└── 系统管理
    ├── 用户管理
    ├── 角色管理
    ├── 菜单管理
    ├── 字典管理
    ├── 系统参数
    ├── 沟通服务协议配置
    ├── 系统配置
    └── 系统日志
```

## 二、不占侧边菜单的入口

| 入口 | 路径 | 备注 |
| :--- | :--- | :--- |
| 登录页 | /login | 独立路由 |
| 个人中心 | Header 头像下拉 | 含我的资料/账号绑定/修改密码 |
| 状态页 | 系统级 | 404/500/无权限/loading |
| 全局导航 | 系统框架 | 顶部栏/侧边栏/面包屑 |

## 三、个人中心入口

个人中心不设侧边菜单，点击右上角头像进入页面后，所有设置项以卡片/列表形式平铺展示，不再依赖下拉隐藏任何功能入口。
