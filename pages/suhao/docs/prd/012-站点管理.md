## 一、模块概述

**模块归属：** 站点中心 → 站点管理

**模块定位：** 管理企业所有销售站点，每个站点是独立的业务来源单元，承载独立的联系方式与运行状态。

**面向角色：** 系统管理员


## 二、页面整体布局

站点管理页面采用 **列表页 + 详情页** 结构：

| 层级 | 内容 |
| :--- | :--- |
| **列表页** | 展示所有站点，含搜索、新增、列表、运行状态 |
| **详情页** | 点击站点行进入，查看该站点基础信息与运行摘要 |


## 三、列表页布局

列表页采用标准企业后台列表布局，从上至下分为三个区域：

| 区域 | 内容 |
| :--- | :--- |
| **筛选区** | 按站点名称或站点编码搜索站点 |
| **工具栏** | 【新增站点】、刷新等操作入口 |
| **表格区** | 站点列表 + 运行状态摘要 |


## 四、顶部操作栏

| 元素 | 说明 |
| :--- | :--- |
| **搜索框** | 按站点名称或站点编码搜索站点 |
| **新增站点** | 点击弹出新增站点弹窗 |


## 五、表格区

### 5.1 表格字段

| 字段 | 说明 |
| :--- | :--- |
| **站点名称** | 站点名称 |
| **站点编码** | 站点唯一编码（不可重复） |
| **域名** | 站点官网地址 |
| **创建时间** | 站点创建时间 |
| **操作** | 编辑 / 启用/停用 |


### 5.2 操作列

| 按钮 | 说明 | 交互 |
| :--- | :--- | :--- |
| **编辑** | 编辑站点基础信息 | 弹出编辑弹窗 |
| **启用/停用** | 切换站点状态 | 点击直接切换；停用需二次确认 |

**停用二次确认弹窗：**

| 元素 | 说明 |
| :--- | :--- |
| 提示文案 | “停用后该站点将不再接收新消息，确认停用？” |
| 确认按钮 | 确认停用 |
| 取消按钮 | 取消操作 |


## 六、新增站点弹窗

**功能定位：** 创建新的销售站点。

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| **站点名称** | 文本输入 | 是 | 站点名称 |
| **站点编码** | 文本输入 | 是 | 唯一编码，不可重复 |
| **域名** | 文本输入 | 是 | 站点官网地址 |
| **联系邮箱** | 文本输入 | 否 | 站点联系邮箱 |
| **WhatsApp** | 文本输入 | 否 | 站点 WhatsApp 账号 |

**校验规则：**

| 校验项 | 规则 | 错误提示 |
| :--- | :--- | :--- |
| 站点名称 | 不可为空 | “请输入站点名称” |
| 站点编码 | 不可为空，且不可重复 | “请输入站点编码” / “该站点编码已存在” |
| 域名 | 不可为空 | “请输入域名” |

**提交逻辑：**
1. 校验站点编码唯一性
2. 创建站点，状态默认为“启用”
3. 关闭弹窗，列表刷新


## 七、编辑站点弹窗

**功能定位：** 修改站点基础信息。

| 字段 | 组件类型 | 是否可编辑 | 说明 |
| :--- | :--- | :--- | :--- |
| 站点名称 | 文本输入 | 是 | 站点名称 |
| 站点编码 | 只读展示 | 否 | 编码不可修改 |
| 域名 | 文本输入 | 是 | 站点官网地址 |
| 联系邮箱 | 文本输入 | 是 | 站点联系邮箱 |
| WhatsApp | 文本输入 | 是 | 站点 WhatsApp 账号 |

**说明：** 编辑弹窗仅修改基础信息。


## 八、运行状态表

**功能定位：** 在列表页快速展示各站点消息拉取与接口运行状态。

| 字段 | 说明 |
| :--- | :--- |
| **站点名称** | 站点名称 |
| **接口状态** | ● 正常 / ● 异常（消息拉取接口状态） |
| **最近拉取时间** | 最近一次消息拉取的时间及拉取条数 |

> 说明：运行状态表仅展示消息拉取状态（运维监控）。


## 九、异常状态与兜底逻辑

| 场景 | 处理方式 |
| :--- | :--- |
| 站点列表无数据 | 显示“暂无站点”占位图，提示“点击【新增站点】创建第一个站点” |
| 搜索无结果 | 显示“未找到匹配的站点”，建议“请调整搜索条件后重试” |
| 数据加载失败 | 显示“数据加载失败，请刷新重试”，提供【重新加载】按钮 |
| 停用站点时存在进行中线索 | 允许停用，停用后该站点不再接收新消息，历史数据保留 |
| 站点编码重复 | 新增/编辑时校验拦截，提示“该站点编码已存在” |


## 十、交互说明汇总

| 交互行为 | 说明 |
| :--- | :--- |
| 搜索站点 | 输入关键词实时过滤列表 |
| 点击新增站点 | 弹出新增站点弹窗 |
| 点击编辑 | 弹出编辑站点弹窗 |
| 点击启用/停用 | 停用需二次确认，启用直接生效 |
| 点击配置 | 跳转至该站点的配置页 |


## 十一、与其他模块的关系

| 关联模块 | 关系说明 |
| :--- | :--- |
| **线索中心** | 站点是线索的来源归属 |
| **客户中心** | 站点是客户的来源归属 |
| **沟通中心** | 站点关联邮箱和 WhatsApp；消息接收依赖站点的联系方式配置 |
| **系统管理** | 站点管理是系统管理员的核心职责之一 |
