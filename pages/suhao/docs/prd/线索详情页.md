## 一、模块概述

**模块归属：** 线索中心 → 线索列表 → 点击线索名称进入

**模块定位：** 展示单条线索的完整信息，是销售跟进、信息查看、状态流转的核心操作页面。

**面向角色：** 销售业务员、销售主管


## 二、页面整体布局

线索详情页面采用 **上下 + 左右混合布局**：

| 区域 | 宽度 | 内容 |
| :--- | :--- | :--- |
| **顶部信息区** | 全宽 | 线索标题、状态、核心属性及操作按钮 |
| **主体内容区** | 全宽，内部两栏 | 左栏（70%）：沟通记录、跟进日志、关联合同；右栏（30%）：AI 智能分析、基础信息、关联对象 |


## 三、顶部信息区

**功能定位：** 展示线索核心摘要，提供关键状态信息和主要操作入口。

### 3.1 顶部字段

| 字段 | 说明 | 展示规则 |
| :--- | :--- | :--- |
| **线索名称** | 默认企业名称 | 大号字体显示，位于顶部左上方 |
| **线索编号** | 系统唯一编号 | 以标签形式展示在名称右侧 |
| **状态** | 线索当前状态 | 状态值：公海待分配、已分配、跟进中、高意向、已转客户、无效、丢失，以彩色徽章展示 |
| **阶段** | 最新跟进阶段 | 展示最新跟进记录的阶段值，位于状态右侧 |
| **负责人** | 当前线索负责人 | 展示负责人姓名及头像 |
| **来源站点** | 线索来源站点 | 展示站点名称 |
| **来源渠道** | 线索来源渠道 | 展示渠道名称（官网询盘 / WhatsApp / 展会等） |
| **创建时间** | 线索创建时间 | 展示完整日期时间 |

### 3.2 顶部操作按钮

| 按钮 | 说明 | 前置条件 |
| :--- | :--- | :--- |
| **录入跟进** | 新增跟进记录 | 无条件 |
| **转客户** | 将线索转为客户 | 线索状态为“高意向” |
| **编辑** | 编辑线索基础信息 | 无条件 |
| **更多** | 下拉菜单 | 包含：【修改状态】、【无效】、【丢失】 |

> 更多下拉说明：
> - 【修改状态】：弹出状态修改下拉选择框，可手动调整线索状态（受权限控制）
> - 【无效】：将线索标记为“无效”状态
> - 【丢失】：将线索标记为“丢失”状态


## 四、左栏：主体内容区

**功能定位：** 展示线索的沟通记录、跟进日志及关联合同。

### 4.1 标签页切换

左栏采用标签页切换方式，包含以下三个标签页：

| 标签页 | 说明 |
| :--- | :--- |
| **沟通记录** | 展示该线索关联的所有邮件和 WhatsApp 消息 |
| **跟进日志** | 展示该线索的全部跟进记录 |
| **关联合同** | 展示该线索关联的全部合同 |

### 4.2 沟通记录

**功能定位：** 展示线索关联的所有沟通消息，统一查看客户互动历史。

**展示内容：**

| 元素 | 说明 |
| :--- | :--- |
| 时间线 | 按时间倒序排列所有消息和会话 |
| 消息类型标识 | 邮件和 WhatsApp 用不同图标区分 |
| 消息摘要 | 展示消息主题或正文摘要，点击可跳转至沟通中心查看完整内容 |
| 收发方向 | 区分收到的客户消息和发出的回复 |

**筛选：**
- 顶部支持按渠道筛选：全部 / 邮件 / WhatsApp
- 支持按时间范围筛选

**交互规则：**
- 点击单条消息记录，跳转至沟通中心对应邮件详情或 WhatsApp 会话详情
- 消息列表为空时，显示“暂无沟通记录”

### 4.3 跟进日志

**功能定位：** 展示该线索的全部跟进记录，完整还原销售过程。

**表格字段：**

| 字段 | 说明 |
| :--- | :--- |
| 跟进时间 | 跟进记录的创建时间 |
| 跟进人 | 创建跟进记录的用户 |
| 跟进方式 | 电话 / 邮件 / WhatsApp / 会议 / 拜访 / 备注 |
| 阶段 | 跟进时记录的阶段值 |
| 跟进内容 | 跟进记录的详细内容 |
| 下次跟进时间 | 计划的下次跟进时间 |

**交互规则：**
- 按跟进时间倒序排列
- 支持分页展示（默认每页 20 条）
- 跟进记录不可修改、不可删除
- 点击【录入跟进】按钮，弹出跟进记录录入弹窗，与列表页交互一致

### 4.4 关联合同

**功能定位：** 展示该线索关联的全部合同。

**表格字段：**

| 字段 | 说明 |
| :--- | :--- |
| 合同编号 | 系统生成的合同编号，点击可跳转至合同详情 |
| 合同名称 | 合同名称 |
| 客户名称 | 关联的客户名称 |
| 金额 | 合同金额 |
| 币种 | 合同币种 |
| 状态 | 合同状态：已签约 / 执行中 / 已完成 / 已终止 / 已作废 |
| 签约时间 | 合同签约时间 |

**交互规则：**
- 合同列表为空时，显示“暂无关联合同”
- 点击合同编号跳转至合同详情页


## 五、右栏：信息面板

**功能定位：** 展示 AI 智能分析结果、线索基础信息和关联业务对象。

### 5.1 AI 智能分析区

**功能定位：** 展示 AI 对该线索的分析结果。

#### 1）AI 意向总结

- **展示内容：** AI 生成的意向总结文本，涵盖客户需求、采购动力、销售建议等
- **说明：** 意向总结属于单条线索信息，转客户时不同步至客户
- **交互：** 支持【复制总结】按钮

#### 2）AI 标签

- **展示内容：** AI 自动生成的线索标签
- **展示规则：** 以标签形式水平排列
- **说明：** 此处为线索的 AI 标签，与邮件 AI 标签相互独立

#### 3）意向产品

- **展示内容：** AI 识别的客户感兴趣产品类目
- **示例：** 毛绒玩具、工业零件
- **说明：** 若 AI 尚未识别，显示“未识别”

### 5.2 基础信息区

**功能定位：** 展示线索的基础业务信息。

| 字段 | 说明 |
| :--- | :--- |
| 企业名称 | 线索关联的企业名称 |
| 联系人姓名 | 主要联系人姓名 |
| 联系人邮箱 | 主要联系人邮箱 |
| 联系人电话 | 主要联系人电话 |
| 来源站点 | 线索来源站点 |
| 来源渠道 | 线索来源渠道 |
| 创建人 | 线索创建人 |
| 创建时间 | 线索创建时间 |

### 5.3 关联对象区

**功能定位：** 展示线索关联的业务对象，提供快速跳转。

| 关联对象 | 展示内容 | 交互 |
| :--- | :--- | :--- |
| **关联客户** | 若线索已转客户，展示客户名称和状态 | 点击跳转至客户详情 |
| **关联消息** | 展示关联的邮件和 WhatsApp 消息数量 | 点击跳转至沟通记录标签页 |


## 六、弹窗交互

### 6.1 录入跟进弹窗

| 字段 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| 跟进方式 | 下拉选择 | 是 | 电话 / 邮件 / WhatsApp / 会议 / 拜访 / 备注 |
| 跟进内容 | 多行文本 | 是 | 跟进记录详细内容 |
| 阶段 | 下拉选择 | 是 | 跟进阶段，选项由管理员在后台配置 |
| 下次跟进时间 | 日期+时间 | 否 | 计划下次跟进时间 |

### 6.2 转客户确认弹窗

| 元素 | 说明 |
| :--- | :--- |
| 线索信息 | 展示当前线索的编号和名称 |
| 客户匹配结果 | 系统自动校验当前站点下是否已存在同名客户，展示匹配结果 |
| 确认按钮 | 确认后执行转客户操作（同步 AI 信息、归集联系人并去重合并） |
| 取消按钮 | 关闭弹窗，不执行操作 |

### 6.3 编辑线索弹窗

| 字段 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| 企业名称 | 文本输入 | 是 | 线索关联的企业名称 |
| 联系人姓名 | 文本输入 | 否 | 主要联系人姓名 |
| 联系人邮箱 | 文本输入 | 否 | 主要联系人邮箱 |
| 联系人电话 | 文本输入 | 否 | 主要联系人电话 |
| 来源站点 | 下拉选择 | 是 | 线索来源站点 |
| 来源渠道 | 下拉选择 | 否 | 线索来源渠道 |

### 6.4 修改状态弹窗

| 元素 | 说明 |
| :--- | :--- |
| 当前状态 | 展示线索当前状态（只读） |
| 目标状态 | 下拉选择目标状态（可选：跟进中、高意向、无效、丢失） |
| 变更原因 | 多行文本，必填，记录状态变更原因 |
| 确认按钮 | 确认后变更状态并自动生成跟进记录 |

> 说明：状态变更自动生成一条跟进记录，跟进方式为“备注”，跟进内容为状态变更说明。


## 七、异常状态与兜底逻辑

| 场景 | 处理方式 |
| :--- | :--- |
| 线索不存在 | 显示“线索不存在或已被删除”，提供返回列表按钮 |
| 数据加载失败 | 显示“数据加载失败，请刷新重试”，提供【重新加载】按钮 |
| AI 服务不可用 | AI 智能分析区显示灰色占位符，提示“AI 服务暂时不可用” |


## 八、交互说明汇总

| 交互行为 | 说明 |
| :--- | :--- |
| 点击录入跟进 | 弹出跟进记录录入弹窗 |
| 点击转客户 | 弹出转客户确认弹窗（仅高意向状态显示） |
| 点击编辑 | 弹出编辑线索弹窗 |
| 点击更多→修改状态 | 弹出状态修改弹窗 |
| 点击更多→无效 | 二次确认后标记为无效，自动生成跟进记录 |
| 点击更多→丢失 | 二次确认后标记为丢失，自动生成跟进记录 |
| 点击沟通记录条目 | 跳转至沟通中心对应消息详情 |
| 点击关联合同编号 | 跳转至合同详情页 |
| 点击关联客户名称 | 跳转至客户详情页（仅已转客户时显示） |
| 切换沟通记录筛选渠道 | 刷新沟通记录列表 |
| 切换标签页 | 切换左栏展示内容（沟通记录 / 跟进日志 / 关联合同） |
