## 一、模块概述

**模块归属：** 客户中心 → 客户列表

**模块定位：** 统一管理所有客户档案，是企业客户资产的集中管理页面。

**面向角色：** 销售业务员、销售主管、区域负责人


## 二、页面整体布局

客户列表页面采用 **单页表格布局**，从上至下分为三个区域：

| 区域 | 内容 |
| :--- | :--- |
| **筛选条件栏** | 多维度筛选条件，支持精细化查询 |
| **表格区** | 客户数据表格，含操作按钮及分页 |


## 三、筛选条件栏

**功能定位：** 提供多维度筛选条件，支持精细化查询。

| 筛选项 | 类型 | 说明 | 默认值 |
| :--- | :--- | :--- | :--- |
| **客户关键词** | 文本输入 | 模糊搜索客户名称、客户编号 | 空 |
| **国家/地区** | 下拉选择 | 按客户所在国家/地区筛选 | 全部 |
| **行业** | 下拉选择 | 按客户所属行业筛选 | 全部 |
| **负责人** | 下拉选择 | 按客户负责人筛选，可选当前用户有权限查看的团队成员 | 全部 |
| **站点** | 下拉选择 | 按来源站点筛选 | 全部 |

**操作按钮：**

| 按钮 | 说明 |
| :--- | :--- |
| **查询** | 按当前筛选条件刷新表格数据 |
| **重置** | 将所有筛选条件恢复为默认值 |
| **导出客户** | 将当前筛选结果导出为 CSV 文件，导出范围为当前筛选条件下的全部数据（不限当前页），受操作权限控制 |
| **高级筛选** | 展开更多筛选条件 |


## 四、表格区

### 4.1 顶部操作按钮

位于表格左上方，提供客户操作入口：

| 按钮 | 说明 | 前置条件 |
| :--- | :--- | :--- |
| **新建客户** | 弹出新建客户弹窗，直接创建客户 | 无条件 |
| **客户转移** | 将选中客户的负责人转移给其他销售 | 至少选中一条客户 |

### 4.2 表格字段

| 字段 | 说明 | 展示规则 |
| :--- | :--- | :--- |
| **复选框** | 用于选中单条或多条客户 | 位于表格最左侧 |
| **客户名称** | 客户企业名称 | 点击可跳转至客户详情页 |
| **来源站点** | 客户所属站点（聚合该客户下所有线索的来源站点，去重展示） | 展示站点名称，多个站点用顿号分隔 |
| **国家/地区** | 客户所在国家 | 展示国家名称 |
| **负责人** | 客户当前负责人 | 展示负责人姓名 |
| **客户标签** | 客户级别独立维护的标签 | 多个标签水平排列，超出 3 个显示“+N” |
| **关联线索数** | 该客户下关联的线索数量 | 展示数字，点击可跳转至线索列表（按该客户过滤） |
| **最近签约合同** | 最近一次签约合同的时间和合同编号 | 展示格式：合同编号（签约时间），如“CON-2026-0081（2026-06-16）” |
| **操作** | 单条客户操作入口 | 包含：【客户详情】、【线索记录】、【AI画像】、【更多】下拉 |


### 4.3 操作列详细说明

| 按钮 | 说明 | 前置条件 |
| :--- | :--- | :--- |
| **客户详情** | 跳转至客户详情页 | 无条件 |
| **线索记录** | 跳转至线索列表，自动按该客户过滤 | 无条件 |
| **AI画像** | 弹出 AI 画像弹窗，展示从线索同步的企业信息 | 无条件 |
| **更多** | 下拉菜单，包含：【编辑】、【转移】 | 编辑、转移受权限控制 |


## 五、新建客户弹窗

**功能定位：** 支持销售直接创建客户，不依赖线索转化。

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| **客户名称** | 文本输入 | 是 | 客户企业名称 |
| **国家/地区** | 下拉选择 | 否 | 客户所在国家 |
| **行业** | 下拉选择 | 否 | 客户所属行业 |
| **联系人姓名** | 文本输入 | 否 | 主要联系人姓名 |
| **联系人职位** | 文本输入 | 否 | 主要联系人职位 |
| **联系人邮箱** | 文本输入 | 否 | 主要联系人邮箱 |
| **联系人电话** | 文本输入 | 否 | 主要联系人电话 |
| **来源站点** | 下拉选择 | 是 | 客户所属站点 |

**提交逻辑：**
1. 校验客户名称不可为空
2. 校验来源站点不可为空
3. 提交后创建客户，当前用户自动成为客户负责人
4. 联系人信息写入客户下的联系人对象
5. 自动生成一条跟进记录（跟进方式：备注，内容：“新建客户”）
6. 客户状态默认为“潜在客户”


## 六、AI 画像弹窗

**功能定位：** 展示从线索同步至客户的企业信息（BR-036）。

| 展示内容 | 说明 |
| :--- | :--- |
| **企业画像** | AI 提取的企业画像信息 |
| **企业信息** | 企业名称、行业、年营收、员工规模、总部所在地等 |
| **风险信息** | AI 识别的企业风险信息 |
| **AI 标签** | AI 生成的客户标签（与客户手动标签独立，仅展示不可编辑） |

> 说明：AI 画像数据来源于线索转客户时同步的 AI 信息，为只读展示，不可编辑。


## 七、客户转移（转客户）

**功能定位：** 将客户的负责人从当前负责人变更为其他销售人员，后续该客户新生成的所有线索自动分配给新负责人。

**触发入口：**
- 表格顶部【客户转移】按钮（批量）
- 操作列【更多 → 转移】（单条）

**执行权限：** 当前客户负责人本人 或 销售主管

### 7.1 弹窗字段

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| 客户名称 | 只读展示 | — | 展示当前客户名称 |
| 当前负责人 | 只读展示 | — | 展示当前客户负责人 |
| 新负责人 | 下拉选择 | 是 | 可选当前站点下所有销售人员 |
| 转移备注 | 多行文本 | 是 | 记录转移原因，必填 |

### 7.2 转移逻辑

1. 选择新负责人后，点击确认执行转移
2. 转移成功后：
   - 客户负责人更新为新负责人
   - 该客户后续新生成的所有线索自动分配给新负责人
   - 历史线索负责人保持不变（不因本次转移而变更）
   - 自动生成跟进记录（跟进方式：备注，内容：“客户 [客户名称] 由 [原负责人] 转移至 [新负责人]”）
3. 转移失败时，弹窗提示错误原因

### 7.3 批量转移

- 选中多条客户，点击顶部【客户转移】按钮
- 统一选择新负责人，所有选中客户一并转移
- 转移备注共用


## 八、编辑客户

**功能定位：** 编辑客户下的联系人信息，支持增删改联系人。

**触发入口：** 操作列【更多 → 编辑】

**执行权限：** 当前客户负责人本人 或 销售主管

### 8.1 弹窗功能

| 区域 | 说明 |
| :--- | :--- |
| **客户基础信息** | 只读展示，不可修改（客户名称、国家、行业、来源站点等） |
| **联系人列表** | 展示该客户下的所有联系人 |
| **新增联系人** | 点击新增一行空白联系人表单（姓名、职位、邮箱、电话、是否主要联系人） |
| **编辑联系人** | 点击联系人行上的编辑按钮，字段变为可编辑状态 |
| **删除联系人** | 点击联系人行上的删除按钮，二次确认后删除（逻辑删除） |

**联系人表单字段：**

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| 姓名 | 文本输入 | 是 | 联系人姓名 |
| 职位 | 文本输入 | 否 | 联系人职位 |
| 邮箱 | 文本输入 | 否 | 联系人邮箱 |
| 电话 | 文本输入 | 否 | 联系人电话 |
| 是否主要联系人 | 开关/复选框 | 否 | 标识该联系人是否为主要联系人 |

> 客户本身不可删除，仅支持联系人维度的编辑操作（BR-043）。新增/编辑/删除联系人后自动生成跟进记录。


## 九、异常状态与兜底逻辑

| 场景 | 处理方式 |
| :--- | :--- |
| 客户列表无数据 | 显示“暂无客户”占位图 |
| 搜索无结果 | 显示“未找到匹配的客户”，建议“请调整筛选条件后重试” |
| 数据加载失败 | 显示“数据加载失败，请刷新重试”，提供【重新加载】按钮 |
| 新建客户名称重复 | 校验当前站点下是否已存在同名客户，若存在提示“该客户已存在，请勿重复创建” |
| 客户转移时新负责人与当前相同 | 提交时校验拦截，提示“新负责人与当前负责人相同，无需转移” |
