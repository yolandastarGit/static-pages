## 一、模块概述

**模块归属：** 线索中心 → 线索列表

**模块定位：** 管理当前用户有权限查看的所有线索，是销售跟进工作的核心入口。

**面向角色：** 销售业务员、销售主管


## 二、页面整体布局

线索列表页面采用 **单页表格布局**，从上至下分为三个区域：

| 区域 | 内容 |
| :--- | :--- |
| **顶部状态标签页** | 快捷筛选标签页，点击按状态过滤线索列表 |
| **筛选条件栏** | 多维度筛选条件，支持精细化查询 |
| **表格区** | 线索数据表格，含操作按钮及分页 |


## 三、顶部状态标签页

**功能定位：** 提供按状态快捷筛选的入口，帮助销售快速定位不同阶段的线索。

| 标签页 | 筛选逻辑 |
| :--- | :--- |
| **全部** | 展示当前用户权限范围内的所有线索 |
| **待跟进** | 线索状态为"已分配" |
| **跟进中** | 线索状态为"跟进中"或"高意向" |
| **已成交客户** | 线索状态为"已成交" |
| **无效** | 线索状态为"无效" |
| **丢失** | 线索状态为"丢失" |

**交互规则：**
- 点击标签页，表格数据按对应逻辑刷新
- 当前选中标签页呈现高亮/下划线选中态
- 标签页与筛选区"状态"下拉联动：点击标签页，状态下拉同步切换为对应值
- 其余筛选条件保持不变


## 四、筛选条件栏

**功能定位：** 提供多维度筛选条件，支持精细化查询。

### 4.1 筛选字段

| 筛选项 | 类型 | 说明 | 默认值 |
| :--- | :--- | :--- | :--- |
| **站点** | 下拉选择 | 按站点过滤线索，可选范围为用户已授权的站点 | 全部可见站点 |
| **创建时间** | 日期区间 | 按线索创建时间范围筛选 | 本月 |
| **线索关键词** | 文本输入 | 模糊搜索线索编号、线索联系人、客户企业名称、邮箱 | 空 |
| **状态** | 下拉选择 | 按线索状态筛选 | 全部 |
| **阶段** | 下拉选择 | 按跟进阶段筛选，选项取自后台配置的完整阶段列表 | 全部 |
| **意向产品** | 下拉选择 | 按 AI 识别的客户感兴趣产品类目筛选，选项来源于系统中所有线索已有的产品类目集合，支持输入检索 | 全部 |

### 4.2 操作按钮

| 按钮 | 说明 |
| :--- | :--- |
| **查询** | 按当前筛选条件刷新表格数据 |
| **重置** | 将所有筛选条件恢复为默认值 |
| **导出** | 将当前筛选结果导出为 CSV 文件，导出范围为当前筛选条件下的全部数据（不限当前页），受操作权限控制。导出失败时提示"导出失败，请重试" |
| **高级筛选** | 展开更多筛选条件 |

**高级筛选（展开后）：**

| 筛选项 | 类型 | 说明 |
| :--- | :--- | :--- |
| **来源渠道** | 下拉选择（多选） | 选项：官网询盘、自然询盘、WhatsApp、展会、客户转介绍、其他 |
| **负责人** | 下拉选择 | 可选范围为当前用户有权限查看的团队成员列表 |
| **标签** | 下拉选择（多选） | 支持按标签筛选，多选时逻辑为"包含任一标签"（OR） |


## 五、表格区

### 5.1 顶部操作按钮

位于表格左上方，提供线索操作入口：

| 按钮 | 说明 | 前置条件 |
| :--- | :--- | :--- |
| **录入跟进** | 为选中的单条线索新增跟进记录 | 仅支持单条线索。若选中多条线索，按钮置灰并提示"录入跟进仅支持单条线索" |
| **跟进日志** | 跳转至独立的跟进日志列表页面，展示所有线索的全部跟进记录（含筛选、排序、分页功能） | 无条件 |
| **转客户** | 将选中的线索转为客户 | 至少选中一条线索，且线索状态为"高意向"（手动批量转客户保留，用于主管在特殊场景下强制触发） |

### 5.2 表格字段

| 字段 | 说明 | 展示规则 |
| :--- | :--- | :--- |
| **复选框** | 用于选中单条或多条线索 | 位于表格最左侧 |
| **线索编号** | 系统自动生成的唯一编号 | 固定格式，如 LEAD-2026-0911 |
| **客户名称** | 关联客户名称 | 若线索已关联客户，展示客户名称，点击跳转至客户详情页 |
| **线索联系人** | 线索的发件人或 WhatsApp 联系对象 | 显示联系人姓名。若联系人已归集至客户，增加显示客户名称。姓名缺失时显示"匿名联系人" |
| **来源** | 线索来源渠道 | 系统根据来源消息渠道自动识别；若无法识别，默认"其他"。取值：官网询盘、自然询盘、WhatsApp、展会、客户转介绍、其他 |
| **负责人** | 当前线索负责人姓名 | 展示负责人姓名 |
| **状态** | 线索当前状态 | 状态值：公海待分配、已分配、跟进中、高意向、已成交、无效、丢失。以彩色徽章展示。线索成交后，状态锁定为"已成交"不可变更，但仍允许录入跟进和更新阶段 |
| **阶段** | 跟进阶段（取自最新跟进记录） | 取值示例：待首响、已联系、样品阶段、需求确认、合同已成交。无跟进记录时显示"-" |
| **标签** | AI 标签 + 手动标签混合展示 | 多个标签水平排列，AI 标签与手动标签用不同样式区分 |
| **意向产品** | AI 识别的客户感兴趣产品类目 | 展示前 2 个产品类目，超出 2 个时显示"+N"（如"毛绒玩具、工业零件 +2"），悬停展示全部类目。未识别时显示"-" |
| **最近跟进** | 最近一次跟进记录的创建时间 | 展示时间，无跟进记录时显示"-" |
| **创建时间** | 线索创建时间 | 展示完整日期时间 |
| **操作** | 单条线索操作入口 | 包含：【跟进】、【打标】、【编辑】、【更多】下拉 |

### 5.3 操作列详细说明

| 按钮 | 说明 |
| :--- | :--- |
| **跟进** | 快捷录入跟进记录，点击弹出跟进录入弹窗，预填当前线索 |
| **打标** | 打开标签管理弹窗，支持新增/删除标签（含 AI 标签和手动标签） |
| **编辑** | 打开线索编辑弹窗，修改线索基础信息 |
| **更多** | 下拉菜单，包含：【查看详情】、【查看邮件】、【查看 WhatsApp】、【修改状态】 |

**查看邮件 / 查看 WhatsApp 跳转行为：**
- 若线索关联了邮件/WhatsApp 消息：跳转至对应模块，自动筛选该线索关联的消息
- 若线索无关联消息（如展会来源）：跳转至对应模块首页，不附加筛选条件

### 5.4 分页

| 配置项 | 说明 |
| :--- | :--- |
| 默认每页条数 | 50 条 |
| 每页条数选项 | 20 / 50 / 100 |
| 跳转 | 支持输入页码跳转 |


## 六、标签展示规则

**标签类型：**

| 标签类型 | 说明 | 样式 |
| :--- | :--- | :--- |
| AI 标签 | AI 自动生成的标签（如"高增长潜力"、"复购询盘"） | 蓝色底 + 圆角 + AI 图标 |
| 手动标签 | 用户手动添加的标签（如"欧洲市场"、"老客户介绍"） | 灰色底 + 圆角 |

**展示规则：**
- 每个线索最多展示 3 个标签
- 超过 3 个时，第 3 个标签显示为"+N"形式（如"+2"），悬停显示全部标签
- 点击标签可快速筛选包含该标签的线索：等价于筛选区"标签"下拉选中该标签（单选），其余筛选条件不变。若筛选区已选中多个标签，点击标签则替换为当前标签（单选覆盖）
- 若筛选区"标签"多选筛选时，逻辑为**包含任一标签（OR）**


## 七、弹窗交互

### 7.1 录入跟进弹窗

| 字段 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| 跟进方式 | 下拉选择 | 是 | 选项：电话、邮件、WhatsApp、会议、拜访、备注 |
| 跟进内容 | 多行文本 | 是 | 跟进记录详细内容 |
| 客户关注 | 多选（Checkbox 组） | 否 | 选项由后台配置，如：报价/价格、交期、资质证书、产品定制能力、产品目录册、项目案例、求代理商合作、样品费及归属、是否为工厂、支付方式/条件、咨询起订量(MOQ) |
| 跟进附件 | 文件上传 | 否 | 支持上传图片、文档等，MVP 支持常见文件格式 |
| 阶段 | 下拉选择 | 是 | 跟进阶段，选项由管理员在后台配置。若后台未配置任何阶段，阶段降级为非必填，仍允许保存跟进记录，同时弹窗提示"暂无跟进阶段配置，请联系管理员设置" |
| 下次跟进时间 | 日期+时间 | 否 | 计划下次跟进时间 |

### 7.2 转客户确认弹窗

| 元素 | 说明 |
| :--- | :--- |
| 线索信息 | 展示当前线索的编号和名称 |
| 客户匹配结果 | 系统自动校验当前站点下是否已存在同名客户。匹配维度：客户名称（完全匹配为主，模糊匹配为辅）；匹配范围：仅限当前线索所属站点内的已有客户。<br>• 匹配到 0 个客户 → 展示"未发现匹配客户"，将直接创建新客户<br>• 匹配到 1 个客户 → 展示匹配结果，用户确认后关联<br>• 匹配到多个客户 → 列表展示所有匹配结果，由用户选择关联目标<br>• 用户也可忽略匹配结果，强制创建新客户 |
| 确认按钮 | 确认后执行转客户操作 | 系统执行转客户流程，匹配并关联已有客户或创建新客户。新创建的客户归属线索的来源站点 |
| 取消按钮 | 关闭弹窗，不执行操作 |

### 7.3 打标签弹窗

| 元素 | 说明 |
| :--- | :--- |
| 已有标签 | 展示当前线索已拥有的全部标签（AI 标签 + 手动标签） |
| 标签输入 | 输入框，输入后回车生成新标签 |
| 推荐标签 | 展示 AI 推荐标签，点击快速添加 |
| 删除 | 手动标签可删除，AI 标签不可删除（由系统管理） |


## 八、异常状态与兜底逻辑

| 场景 | 处理方式 |
| :--- | :--- |
| 列表无数据 | 显示"暂无线索"占位图，提示"当前筛选条件下没有匹配的线索"，并提供引导链接【前往公海池认领线索】（跳转至线索中心→公海池） |
| 搜索无结果 | 显示"未找到相关线索"，建议"请调整筛选条件后重试" |
| 数据加载失败 | 显示"数据加载失败，请刷新重试"，提供【重新加载】按钮 |
| 批量转客户前置校验 | 校验所有选中线索状态是否为"高意向"，若有不符合条件的，弹窗提示并阻止操作 |
| 批量转客户部分成功 | 逐条执行转客户操作。<br>• 全部成功 → 提示"成功转客户 N 条"<br>• 部分成功 → 弹窗提示"成功转客户 M 条，失败 N 条"，展示失败原因列表<br>• 全部失败 → 提示"转客户失败"，展示错误原因 |

## 十、跨模块协同

### 10.1 查看邮件

**触发：** 线索列表 → 操作列【更多】→ 【查看邮件】

**自动筛选逻辑：**
- 跳转至沟通中心 → 邮件页
- 自动带入筛选条件：当前线索关联的发件人邮箱
- 邮件列表自动筛选出与该发件人的全部往来邮件

**回退机制：**
- 若该发件人无关联邮件，跳转至邮件页默认收件箱，Toast 提示"该线索暂无关联邮件，已为您打开默认收件箱"
- 若线索关联邮箱账号未配置，Toast 提示"该线索关联邮箱账号未配置，请先在系统管理中配置"

### 10.2 查看 WhatsApp

**触发：** 线索列表 → 操作列【更多】→ 【查看 WhatsApp】

**自动筛选逻辑：**
- 跳转至沟通中心 → WhatsApp 页
- 自动带入筛选条件：当前线索关联的手机号
- 会话列表自动筛选出与该手机号的 WhatsApp 对话

**回退机制：**
- 若该手机号无关联会话，跳转至 WhatsApp 默认会话列表，Toast 提示"该线索暂无 WhatsApp 关联会话"
- 若该线索未填写手机号，操作按钮置灰并显示"该线索未填写手机号"


## 十一、批量操作扩展

### 11.1 批量打标

**触发：** 列表多选 → 顶部【批量打标】

**弹窗字段：**

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :---: | :--- |
| 已选线索数 | 只读展示 | — | 展示已选中的线索数量 |
| 标签输入 | 标签输入框 | 是 | 输入后回车生成标签，支持多标签 |
| 标签来源 | 单选 | 是 | 手动输入 / AI 推荐标签 |
| 是否覆盖现有标签 | 开关 | 否 | 关闭：追加新标签；开启：替换全部现有手动标签 |
| 影响范围 | 只读展示 | — | "本次操作将影响 N 条线索，覆盖现有 M 个标签" |

**校验规则：**
- 标签不可为空，至少 1 个
- 单个标签最长 20 字符
- 单次批量打标最多 100 条

**结果反馈：**
- 全部成功：Toast 提示"成功为 N 条线索打标"
- 部分成功：弹窗展示成功/失败明细
- 全部失败：弹窗展示失败原因列表

### 11.2 批量修改状态

**触发：** 列表多选 → 顶部【批量修改状态】

**弹窗字段：**

| 字段 | 组件类型 | 是否必填 | 说明 |
| :--- | :--- | :---: | :--- |
| 已选线索数 | 只读展示 | — | 展示已选中的线索数量 |
| 目标状态 | 下拉选择 | 是 | 仅展示当前可流转状态（按 BR-XXX 状态机校验） |
| 变更原因 | 多行文本 | 否 | 可选记录批量变更原因 |
| 影响范围 | 只读展示 | — | "本次操作将影响 N 条线索" |

**校验规则：**
- 状态变更需符合各线索的当前状态机（不符合的线索单独标注并跳过）
- 单次批量修改最多 100 条
- 状态变更后自动生成跟进记录（跟进方式 = 状态变更）

**结果反馈：**
- 全部成功：Toast 提示"成功修改 N 条线索状态"
- 部分失败：弹窗展示不符合状态机的线索列表
- 全部失败：弹窗展示失败原因

### 11.3 批量导出

**触发：** 列表多选 → 顶部【批量导出】

**弹窗字段：**

| 字段 | 组件类型 | 是否必填 | 默认值 | 说明 |
| :--- | :--- | :---: | :--- | :--- |
| 导出范围 | 单选 | 是 | 当前选中 | 当前选中 / 当前筛选结果 / 当前页 |
| 导出列 | 多选下拉 | 是 | 全部 | 至少 1 列，至多全部 |
| 文件格式 | 单选 | 是 | Excel | Excel / CSV |
| 字段映射 | 下拉选择 | 否 | 系统默认 | 选择"系统默认"使用系统字段名；选择"自定义"允许调整列名 |
| 导出上限 | 只读展示 | — | — | "本次导出最多 10000 条" |

**导出规则：**
- 超过 10000 条提示"数据量过大，请缩小筛选范围"
- 异步导出：提交后 Toast 提示"导出任务已提交"，完成后在站内信通知
- 导出记录：系统日志记录"线索导出"操作（操作人/范围/数量）
