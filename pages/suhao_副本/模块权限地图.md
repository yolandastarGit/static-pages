## 5. Agent 分工建议

### 可以独立 Agent 开发（受限并行）

| Agent | 范围 | 并行条件 |
|-------|------|----------|
| **Contracts Agent** | `contract-list`, `contract-customer`, contracts drawer/modal | 边界最清晰，跨模块依赖少 |
| **Site Agent** | 站点 4 页 + site drawer/modal | 相对独立，但共享 `system.html` |
| **Leads Agent** | 线索 4 页 + leads drawer + 线索 modal | 可并行，但 AI/客户/沟通交叉需确认 |
| **Customers Agent** | 客户 2 页 + 3 Tab + customers drawer | 可并行，360 抽屉跨域多 |
| **Communication Agent** | 沟通 4 页 + 邮件/WhatsApp | 可并行，来件转线索需确认 |
| **Analytics Agent** | 10 个分析页 | 只读依赖，可并行，但不能改 datasets 结构 |
| **Workbench Agent** | workbench + message-center | 可并行，跨域跳转少 |

### 必须由主 Agent 管理

| 管理项 | 原因 |
|--------|------|
| **`router.js` 全部** | 路由/状态/登录基础设施 |
| **`PAGE_MODULE_MAP` / `navGroups` / `navRedirects`** | pageId 注册表，新增页面必经此处 |
| **`renderPage()` / `renderCustom()` / `panelFns`** | 页面分发中枢 |
| **`openDrawer()` / `openModal()` 入口** | 中央分发，改签名影响全局 |
| **RBAC 配置**（`roleMenuAccess`, `rolePageMode`, `roleScopes`） | 全局权限门禁 |
| **6 个 HTML 公共 shell** | 结构一致性问题 |
| **`datasets` 结构变更审批** | 数据契约，影响跨模块 |
| **新增/删除 pageId** | 需同步 router + pageMeta + RBAC + HTML 入口 |
| **AI Agent 的 `canUseAiFeature()` 变更** | 全局 AI 门禁 |
| **UI Agent 的公共组件 API 变更** | 影响所有 render 函数 |

### 建议锁定目录/文件

```
🔒 完全锁定（仅主 Agent）:
  assets/router.js
  index.html（公共 shell 部分）
  *-module.html 的公共 DOM 结构（topbar/sidebar/drawer/modal）
  shared.js: L1-450（nav/RBAC/AI config）
  shared.js: renderPage 相关（renderCustom/panelFns 映射表）
  shared.js: openDrawer/openModal 入口函数

⚠️ 按 pageId 区间锁定（子 Agent 仅改指定函数）:
  shared.js: 各 render*Page() 函数体
  shared.js: drawerDetail.{本模块type}
  shared.js: modalContent() 中本模块 type 分支
  shared.js: datasets.{本模块key}

📋 需确认后改:
  shared.js: datasets 字段结构
  styles.css: 全局 layout 类（.topbar/.sidebar/.layout）
  任何 nav() 新增目标 pageId
```

## 6. 结论

当前项目 **pageId 边界清晰，但文件级隔离缺失**。多 Agent 协作的可行模式是：

**「主 Agent 锁基础设施 + 子 Agent 按 pageId 改 shared.js 指定函数区间」**

在不做文件拆分的前提下，每个子 Agent 的工作范围应严格限定为：

```
ALLOWED:  render{Module}Page() + drawerDetail.{type} + modalContent('{type}') + datasets.{key}
FORBIDDEN: router.js / RBAC / openDrawer入口 / openModal入口 / 公共shell / 其他模块函数
CONFIRM:   跨模块 nav() / datasets字段变更 / AI门禁 / 新增pageId
```
